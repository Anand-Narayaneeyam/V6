The reports generated by the system can be viewed using the controls available in the report viewer. The report can be saved to Excel, PDF or RTF format by clicking on Excel, PDF or RTF button.

Asset Reports are available under Assets menu in Reports

The following Reports are available in Assets Module.

Asset Class List

This report shows the list of Asset Classes with their attributes and whether the attributes are validated or not. The report can be saved to PDF, Excel and Rich Text formats.

Asset List by Floor

This report gives the details of all Assets in the selected floors.

On selecting this report, the list of floors organized by site/building/floor hierarchy is displayed. From this list, select the required site/building/floor and click on Show Report icon. The Asset list by floor report is generated. The report can be saved to PDF, Excel and Rich Text formats.

Asset Details

This report gives the details of all the Assets.

On selecting this menu, the lists of Asset classes are displayed. From this select the required Asset Classes and click on Show Report icon. The Asset Details report is generated. The report can be saved to PDF, Excel and Rich Text formats.

Asset List by Class

This report gives the list of Assets by class.

On selecting this menu, the lists of Asset classes are displayed. From the list select the Asset classes and click Show Report icon . The Asset list by class report is generated. The report can be saved to PDF, Excel and Rich Text formats.
