This option allows the administrator to view/edit the sites that are added through Administration module.

All sites will be listed with the Site Name, Site Code, Site Status, Building Count, Gross Area etc. Any additional data fields added for sites will also be displayed. The list can be sorted by clicking the column headers.

Edit

This option is for editing the site details.

Select the site to be edited and click on the Edit icon. The details will be displayed in a new window. Make necessary changes and click the Save Changes button.

Attachments

Administrators can view or download the attachments added to a site using this option. Select the required site and click on Attachments icon. Any attachments added to the site will be listed. The attachment can be viewed or downloaded by clicking the View or Download icon.

Keyword Search

This option helps users to search for Sites using keywords. The keyword can be part of Site name, Site Code, description etc. Enter the keyword in the text box provided and click on the Search icon. All sites matching the keyword will be listed.

A facility user would be able to view those Sites and view/download the attachments of the selected site

Advanced Search

On clicking Advanced Search icon, Advanced Search window will be displayed which allows the user to search for the Site details for specific data in known data fields. The data being searched for can be selected from the list boxes or entered into text boxes provided against the data fields. On clicking the Search button, all Sites meeting the search criteria will be listed.

Fields to be displayed in the Search window can be set using the Configure icon provided in the Search window.

Search criteria entered in the Advanced Search window can be cleared by clicking the Clear button provided in the Search window. After clearing the Search criteria and click Search button to clear the Search results in the Sites grid.

A facility user would be able to view those Sites and view/download the attachments of the selected site
