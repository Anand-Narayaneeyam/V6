This option is available under As Builts Module icon. When clicking As Builts icon, menu options like Drawings and Data will be displayed in the Left panel.

All drawings will be listed with details of Site, Building, Floor, Category, Type, Description, DWG File, DWG File Size etc. The list can be sorted on any column by clicking the column headers. Edit Description, View, List Revisions, List Markups and Display Settings options are available in the Drawings list page.

Edit Description

Description of a drawing can be edited through this option. Select the required drawing from the list and click Edit Description icon. Edit Drawing Description window will be displayed. Description entered can be edited and the changes can be saved by clicking the Save Changes button.

View

This option is to view the selected drawing in the browser. Users can add comments and markup the drawing with different shapes and save them to the repository.

Select the drawing to be viewed and click View icon. The selected file will be opened up in the Browser.

User can change the view mode to full screen by clicking the Maximize icon provided above the top right corner of the drawing window. User can exit from the full screen to normal view by clicking the Minimize icon provided above the top right corner of the drawing window.

Drawing Details and Markups

Markups and comments that exist in the selected drawing can be viewed using this option. On clicking Drawing Details and Markups icon, a window opens up with the drawing details and list of existing markups and comments of the selected drawing with check box against each markup.

In the Markups section, Administrator can edit markup description, delete or view the selected markup.

Edit: Select the required markup and click Edit icon. Edit Markup Description window will be displayed. Description entered can be edited and the changes can be saved by clicking the Save Changes button.

Delete: Select the required markup and click Delete icon. A confirmation message will be displayed. Click Yes to delete the selected markup file and the markup will be removed from the repository.

View: Select the required markup and click View icon. Then the selected Markup will be displayed in the drawing.

Comment

This option is used to add text comments on the drawing. The comments once added can be moved or deleted. To add text comments on the drawing, click on the Add Comment icon in the toolbar above the drawing. On clicking this icon, the cursor turns to a cross. Click on the drawing where the comment is to be placed, to create a text box. Select the required Font and Font Size. Font Style can be changed to Bold or Italics by clicking on or icon. Enter the comment in the text box that shows up and click Save button. The comments once added can be edited by selecting the comment to be edited after clicking the Edit Comment icon. Edit the changes in the comment in the text box that shows up and click Save button to update the comment. Click the Save icon to save the changes done in the comments in the current Drawing.

Markup

Markups in different shapes (Rectangle, Ellipse, Line, Cloud, Free Hand) can be added in the drawing using this menu. Set the required Line Thickness for the markup. Click on the desired shape icon, the cursor turns to a cross. Click and drag the cursor to make markups anywhere in the drawing. Markups once created cannot be edited, they can be moved or deleted.

Move

Markups and comments can be moved using this option. Click on the Move icon, the cursor changes to a box shape. Select the markup to be moved by bringing the box cursor over the markup and clicking on it. After selection, when the cursor is moved, a rubber band like line will appear showing the line of movement. Click on the location where the Markup is to be placed. The Markup will be moved to the new place.

Delete

Markup or comment can be deleted using this option. Click Delete icon, the cursor changes to a box shape. Click on the Markup to be deleted. A confirmation message will be displayed. Click Yes to the confirmation message, the selected Markup will get deleted.

Save

After creating comments and markups, click on the Save icon to save the markup. A description about the markup also can be entered, and on clicking the Save button the comments and markup added will be saved. On saving the markup each time, it is saved in a different file. All markups are listed and can be viewed through the Drawings: List Markups menu option.

Export

This option is for exporting the current view of the drawing to the user's computer. On clicking Export icon, different file formats like BMP, DWF, DXB, DXF, JPEG, PDF and SVG will be available and the user can select the desired file format to export.

Save As

On clicking Save As icon the current view of the drawing will be saved as a DWG file to a folder in the user's computer.

List Revisions

On clicking List Revisions icon, the revisions added to the drawing will be listed in a grid. Different menu options like Edit Description, Delete and View options are available to manage the revisions.

List Markups

On clicking List Markups icon, all markups available for the drawing will be listed in a grid. Different menu options like Edit, Delete and View options are available to manage the markups.

Display Settings

This option allows the user to change the visibility of the data fields displayed in the data grid. On clicking Display Settings icon, all the available data fields will be displayed here with the System Default Settings with checkboxes against Show in Grid. The desired check boxes can be checked or unchecked and the settings can be saved by clicking the Save Changes button.

Changes made here will be saved as user default settings and will be effective for the logged in user only

For making the display settings as System default, an administrator has to click Yes to the confirmation message "Display Settings updated. Do you want to make this as System default?". Then the display settings will be effective for all the users logged in to that particular Customer.

Download

The DWG files can be downloaded and saved in the user's computer by selecting the drawing and clicking the Download icon.

Keyword Search

This option helps users to search for drawings using keywords. The keyword can be part of drawing file name, drawing description, category, type, site, building, floor etc. Enter the keyword in the text box provided and click on the Search icon. All drawings matching the keyword will be listed.

A facility user would be able to view only those floors/divisions for which privileges are granted to him.

Advanced Search

On clicking Advanced Search icon, Advanced Search window will be displayed which allows the user to search for the drawings details for specific data in known data fields. The data being searched for can be selected from the list boxes or entered into text boxes provided against the data fields. On clicking the Search button, all drawings meeting the search criteria will be listed.

Fields to be displayed in the Search window can be set using the Configure icon provided in the Search window.

Search criteria entered in the Advanced Search window can be cleared by clicking the Clear button provided in the Search window. After clearing the Search criteria and click Search button to clear the Search results in the Drawings grid.

A facility user would be able to view only those floors for which privileges are granted him.
