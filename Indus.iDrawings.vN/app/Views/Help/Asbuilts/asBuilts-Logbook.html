This option allows the Administrator to view log of user activities in the As Builts Module. The period for which the log is to be viewed can be specified, and the users, entities and activities can be selected. This helps to keep track of all the user activities in the module. This option will be available only if the ‘Logbook’ option is enabled by Indus.

To view the log activities in the As Builts Module, user has to select the As Builts Module from the Module list in the Log book Filter option. Log activities can also be filtered based on Users, Entities and Activities.
