As Builts reports are available under Reports icon.

The reports generated by the system can be viewed using the controls available in the report viewer. The report can be saved to Excel, PDF or RTF format by clicking on Excel, PDF or RTF button. The following reports are available in As Builts module.

Sites

This report shows the list of all sites with Site Name, Site Code, Building Count and Status.

Buildings

This report shows the list of all buildings with their site name, building name, floor count, etc.

Floors

This report shows the list of all floors with their site name, building name and floor name.

Drawings - All Drawings

This report shows the detailed list of all drawings.

Drawings - Revisions

This report shows the list of all revisions of drawings.

Drawings - Markups

This report shows the list of all markups of drawings.
