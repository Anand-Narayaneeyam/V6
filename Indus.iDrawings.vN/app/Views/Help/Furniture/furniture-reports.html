The reports generated by the system can be viewed using the controls available in the report viewer. The report can be saved to Excel, PDF or RTF format by clicking on Excel, PDF or RTF button.

Furniture Reports are available under Reports icon.

The following Reports are available in Furniture Module.

Furniture Details

This report gives the details of all the Furniture.

On selecting this menu, the lists of Furniture classes are displayed. From this select the required Furniture Classes and click on Show Report icon. The Furniture Details report is generated. The report can be saved to PDF, Excel and Rich Text formats

Furniture List by Class

This report gives the list of Furniture by class. On selecting this menu, the lists of Furniture classes are displayed. From the list select the Furniture classes and click Show Report icon . The Furniture list by class report is generated.

The report can be saved to PDF, Excel and Rich Text formats.
