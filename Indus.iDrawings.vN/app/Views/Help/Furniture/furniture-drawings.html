All drawings that are unlocked and available for Furniture Management will be listed here, organized by Site, Building and Floor with the Drawing Category, Drawing File Name and File Size. If the option 'Utility Drawing Support' is enabled by Indus, all utility drawings enabled under Drawing Management also will be listed.

View Drawing

On selecting a drawing and clicking on the View icon, the drawing will be opened in a new window. All drawing related operations are provided with a horizontal menu bar.

The Space data and Furniture data are fetched from the database and displayed in the drawing in dynamically created layers. Only the Space Polyline and Space Data layers and any predefined default display layers will be turned on by default.

The Furniture data corresponding to the selected Floor drawing will be displayed on the right side of the Drawing in a tab sheet labeled as ‘Operational’ and ‘Warehoused . All Assigned Furniture will be listed under Operational Tab and all warehoused Furniture if any are also displayed in tab sheet labeled as 'Warehoused'

If the user has no access permission to a Division, the Furniture placed on spaces belonging to that Division will not be displayed in the drawing and data of those furniture also will not be listed.

Right Click Options

On right clicking the mouse on the in the drawing, a pop-up menu will appear with the options like Edit Furniture,Select Multiple Spaces by Window, Pan, Zoom In, Zoom Out, Zoom Window, Zoom Extents and Layers.

Edit Furniture : This option is to edit the Furniture data.

Select Multiple Spaces by window: This option is for selecting multiple spaces in the drawing by selecting individual spaces or by selecting a window for viewing details of the Furniture and space data linked to those spaces.

Pan: This option enables the user to pan the drawing. On selecting this option, the mouse pointer will change to a hand symbol. Now click the left mouse button, and without releasing it, move the mouse to pan the drawing.

Zoom In: This option enables the user to Zoom in the Drawing.

Zoom Out: This option enables the user to zoom out the Drawing.

Zoom Window: This option enables the user to zoom the Drawing to a Window selection.

Zoom Extents: This option enables the user to view the Drawing fully within the drawing display window by resetting any zoomed or panned states.

Layers: This option enables turning On and Off the layers in the base drawing. The dynamically generated layers are not available here. These have to be controlled through the Display Settings. Clicking on the Default button turns on the default space polyline layers.

View Drawing tab is provided with a set of icons namely Distribution Map, Space Tools, Furniture Tools,Tools, Maximize, Export and Save As. All Attachments uploaded with file types – bmp, jpeg etc can be viewed by double clicking the corresponding Furniture or space.

Distribution Map

Distribution Map allows users to get the distribution of space areas across different Organizational Units (Divisions, Departments, Cost Centers etc as defined in the Organizational structure) or based on the Validated Fields. The Spaces belonging to different Organizational Units or the Validated Fields are hatched in the drawing with different colors. Users can select the Organizational Units or the validated fields for which the distribution is required. The color is displayed according to the settings in General Settings: Space: Distribution Map Settings and can be changed by the user if required.

User can be change the color by clicking the Settings icon provided against each Organizational Unit and Validated Field. Settings made here will be applicable for the current instance. A legend based on the distribution map also gets inserted automatically with Color of each Organizational Unit and its Area Sq. Ft.

Space Tools

All the Space related operations such as Edit Space, Move Legend, Select Multiple Spaces by Window, Totalize, Measure Distance and Measure Area are grouped under Space Tools

Edit Space

This option is for editing the Space Data in the Drawings.

Move Legend

This option is to move a legend from one point to another. Select Move Legend option from Space Tools and then click to specify a new location.

Select Multiple Spaces by Window

This option is for selecting multiple spaces in the drawing by selecting individual spaces or by selecting a window for viewing details of the Furniture and space data linked to those spaces.

Click on the Select Multiple Spaces by Window option from Space Tools , a toast message 'Select a window, click to select/deselect space. Right Click to exit' is shown. Then the mouse pointer becomes a cross. Select a window by clicking and dragging inside the drawing. The selected spaces will be hatched. Right Click when all required spaces are selected. Then the Furniture assigned in the selected spaces will be highlighted in the Operational Data gird of Drawing page.

A selected space can be deselected if required, by clicking on the De-Hatch option in the Right Click.

Totalize

This option is to display the Space and Furniture details of the selected Spaces. Select the spaces to be totalized using Select Multiple Spaces by window option and click Totalize option from Space Tools , a message box will be displayed with Space details and Furniture Details of the selected spaces. Apart from this, if any additional data fields of Space are set as Totalizable, the total value of such fields also will be shown. The details shown in Totalize window can be added as legend by clicking the Add Legend button in the Totalize window.

Measure Distance – Point to Point

This option is used to measure the distance between two points. Select Measure Distance - Point to Point option from Space Tools icon and the cursor will turn into a cross. Click close to the wall from which the distance is to be measured. The starting point will snap to the nearest space polyline (Net Polyline for Net only Customers). Then click close to the other wall to which the measurement is to be taken. The end point will snap to the nearest point of the corresponding space polyline/net polyline. The distance between the two walls will be displayed. This marked distance can be removed by clicking the De-hatch option.

Measure Distance – Wall to Wall

This option is used to measure distance between two Walls. Select Measure Distance - Wall to Wall option from Space Tools icon and the cursor will turn into a cross. Click close to the wall from which the distance is to be measured. The starting point will snap to the nearest space polyline (Net Polyline for Net only Customers). Then click close to the other wall to which the measurement is to be taken. The end point will snap to the nearest point of the corresponding space polyline/net polyline. The distance between the two walls will be displayed.

Measure Area

This option is used to measure the area of a polygon in a Drawing. To measure an area, Select Measure Area option from Space Tools and the cursor will turn to a cross. Mark the area to be measured by clicking on any one vertex point, and then clicking on the remaining vertex points one after the other. After the last vertex, close the polygon by right clicking in the drawing. The polygon area will be displayed. This marked area can be removed by clicking the De-Hatch option.

Furniture Tools

Warehouse Furniture

This option is to de-link the selected Furniture from the assigned Space. The de-linked Furniture will be moved to the warehoused grid. Only the Administrator can perform this option. On clicking Warehouse Furniture option from Furniture Tools , a toast message 'Select the Furniture to be warehoused' will be displayed and the mouse pointer will turn to cross. Select the Furniture by clicking on the Furniture symbol. A confirmation message will be displayed. On clicking Yes, the selected Furniture will be de-linked from the drawing.

Move Single Furniture

This option is to move a Furniture from one point to another. Select Move Single Furniture option from Furniture Tools ,a toast message 'Select the Furniture to be moved' will be displayed. Then the mouse pointer will turn to cross and select the Furniture by clicking on it. Then the selected Furniture will be hatched. Click on the exact location to which the Furniture is to be moved.

Move Multiple Furniture

This option is to move multiple Furniture from one space to another. Moving of multiple Furniture to places outside the polyline is not possible through this option. Only Furniture placed in the same space can be moved to another space. Only the Administrator can perform this move operation. Select Move Multiple Furniture option from Furniture Tools , a toast message window will be displayed, prompting 'Click to select/deselect Furniture'. Right click to exit'. Then the mouse pointer becomes a cross. Select the Furniture by clicking on it. On right clicking the mouse after selecting the required Furniture, a rubber band will appear, which will have the first click point as a fixed point, and the free end following the cursor to show the movement direction. Click the desired point to which the move is to be done. The Furniture will be moved based on this move definition. If Some Furniture are moved to different spaces after the Multiple Move operation, a confirmation message will be displayed. On clicking 'Yes' to the message, the Furniture will be moved to the new locations. On clicking 'No', the move operation will get cancelled and the Furniture will be moved back to their original locations.

Tools

All the Drawing related operations such as Display Settings, De-hatch and Stop Blink are grouped under Tools icon.

Display Settings

This optionis for specifying the Common Attributes, Class Attributes and Space Data fields to be displayed in the drawing and the tooltip. Changes made here will be saved as user default settings and will be effective whenever the User opens any drawing.

For making permanent changes, an administrator has to click Yes to the confirmation message "Display Settings updated. Do you want to save this as System default?".

On selecting this option, the common attributes of all Furniture will be listed with check boxes for Drawing and Tooltip. The desired boxes can be checked, and the color of text in the drawing can be changed as required. The changes become effective on clicking the Save Changes button. Changes made here will be saved as User Default Settings and this can be set as System Default settings by clicking Yes to the confirmation message "Display Settings updated. Do you want to set this as System Default?".

The Display Settings for Class specific Attributes of each Furniture Class that are shown in the Drawing and in the Tooltip can be modified using this option. On selecting the required Furniture Class, any class specific attributes defined for the Furniture Class will be listed with check boxes for Drawing and Tooltip. The desired boxes can be checked, and the color of text in the drawing changed as required.

The changes become effective on clicking the Save Changes button. Changes made here will be saved as User Default Settings and this can be set as System Default settings by clicking Yes to the confirmation message "Display Settings updated. Do you want to set this as System Default?".

Space Data

This option is for specifying the Space Data fields, to be displayed in the drawing, and the tooltip. The changes become effective on clicking the Save Changes button. Changes made here will be saved as User Default Settings and this can be set as System Default settings by clicking Yes to the confirmation message "Display Settings updated. Do you want to set this as System Default?".

If a particular data field is to be shown in drawings, check the corresponding check box of the 'Show in Drawing' column.

If a particular data field is to be shown in the Tooltip, check the corresponding check box of the 'Show in Tooltip' column. Click Save Changes button to save the settings.

This option is available only for Architectural drawing.

De-Hatch

This option is to de-hatch the hatched Space (if any) in the Drawing.

Stop Blink

This option is to stop the blinking of the hatched Space (if any) in the Drawing.

Maximize

User can change the view mode to full screen by clicking the icon provided above the top right corner of the drawing window. User can exit the full screen to normal view by clicking the icon provided in the drawing toolbar.

Export

This option is for exporting the current view of the drawing to the user’s computer in any of the following formats namely bmp, dwf, svg, pdf etc. The drawing may be exported with the Space data layers, dynamically generated distribution maps, legends and Furniture if any.

Save As

This option is for saving the Drawing into dwg format. Versions of the AutoCAD can be selected while saving the Drawing.

Operational

Furniture assigned in the drawing will be displayed under Operational grid at the right side of the Drawing.

Select the required Field and enter or select the value to be updated to the selected Furniture and click Save Changes button. The changes have to be made selecting one field at a time.

Delete

The File attachments window opens up on selecting Attachments option from More icon showing any files that are already attached to the Furniture .Attachments can now be added to the selected Furniture.

Warranty Details

Users can add/edit/delete/ warranty details to a Furniture using Warranty Details menu from More icon. Select the required Furniture and click the Warranty Details option.

Add: Warranty Details can be added by clicking the Add icon. New Warranty Details window will be displayed. Select a Warranty Type and Start Date, then enter Warranty Period In Months. User can also enter Warranty Certificate Number and Description if needed. The details can be saved by clicking on the Save button.

Edit: Warranty Details can be edited by selecting it and then clicking the Edit icon. The Edit Warranty Details window will be displayed with the saved details. The user can select Warranty Type and Start Date, then modify Warranty Period In Months. User can also enter Warranty Certificate Number and Description if needed. Changes can be saved by clicking the Save Changes button.

Delete: Delete Warranty Details by selecting it and then clicking the Delete icon.

Alert: This option is used to set Warranty Alert against each Warranty details. Select a Warranty Detail and click Alert icon, then Warranty Alert page will be displayed. User can select a Notification Template and then the Email recipients can be selected from iDrawings Users or Conatcts menu. Alert can be saved by clicking Save Alert button.

Keyword Search

This option is available only after clicking the icon.

This option helps users to search for Furniture details using keywords. The keyword could belong to any data field of the Furniture data. The user need not know or specify which field to look in. The Keyword could be any part of the Furniture data viz. Furniture No, Furniture Class Name etc. On entering the keyword and clicking on the Search icon, the data grid will be filtered with Furniture records having data matching the specified keyword. The Furniture displaying in the grid as a result of search operation will blink in the drawing.

A facility user would be able to view only those floors/divisions for which privileges are granted to him.

Advanced Search

This option helps users to search for Furniture details with specific data in known data fields. The data being searched for can be selected from the list boxes or entered into text boxes provided against the data fields. The user can select one or more data fields viz. Furniture No, Furniture Class Name,Status etc or the additional data fields. After entering the search criteria and clicking Search button, the data grid will be filtered with furniture records which satisfy the selection criteria.

Fields to be displayed in the Search window can be set using the Configure icon, provided in the Search window. Search criteria entered in the Search window can be cleared by clicking the Clear button provided in the Search window. After clearing the Search criteria and click Search button to clear the Search Results in the Operational Data grid.

A facility user would be able to view only those floors for which privileges are granted him.
