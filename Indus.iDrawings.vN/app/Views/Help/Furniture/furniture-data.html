This option displays both assigned and warehoused Furniture.

Add

This option is for adding a new Furniture. On selecting Add icon, the Furniture Data Entry window will be displayed, with the predefined data fields and any additional data fields defined by the user. The details of the Furniture can be entered there. On clicking Save button, the Furniture will be added.

If Autonumbering option is not enabled for the Customer, option to enter Furniture Number will be displayed in New Furniture page.

Edit

This option is for editing the details of a Furniture. Select the Furniture to be edited in the Data Grid and then click Edit icon. The details will be displayed. Make necessary changes and click on Save Changes button to save the changes.

Multiple Edit

Multiple records can be edited by selecting the required Furniture using the Shift or Ctrl keys, and then clicking on the Edit icon. A window will be displayed listing the fields that can be edited. Select the required Field and enter or select the value to be updated to the selected Furniture and click Save Changes button. The changes have to be made selecting one field at a time.

Delete

This option is for deleting Furniture. Select the Furniture to be deleted and click Delete icon. A confirmation message will be displayed. On clicking Yes to the confirmation message, the selected Furniture will be deleted.

Display settings

This option is for specifying the Common Attributes, Class Attributes and Space Data fields to be displayed in the data grid. Changes made here will be saved as user default settings.For making permanent changes, an administrator has to click Yes to the confirmation message "Display Settings updated. Do you want to save this as System default?".

Three tabs namely Common Attributes, Class Attributes and Space Data will be displayed.

Common Attributes

The Display Settings for Common Attributes of all Furniture that are shown in the Datagrid can be modified using this option. On selecting this option, the common attributes of all Furniture will be listed with check boxes for Data. The desired boxes can be checked and the changes become effective on clicking the Save Changes button. Changes made here will be saved as User default Settings. This can be set as System Default settings by clicking Yes to the confirmation message "Display Settings updated. Do you want to set this as System Default?".

Class Atributes

The Display Settings for Class specific Attributes of each Furniture Class that are shown in the Datagrid can be modified using this option. On selecting the required Furniture Class, any class specific attributes defined for the Furniture Class will be listed with check boxes for Data. The desired boxes can be checked and the changes become effective on clicking the Save Changes button. Changes made here will be saved as User default Settings. This can be set as System Default settings by clicking Yes to the confirmation message "Display Settings updated. Do you want to set this as System Default?".

Space Data

This option is for specifying the Space Data fields, to be displayed in the Datagrid. All Space fields will be listed with checkboxes. The desired boxes can be checked and the changes become effective on clicking the Save Changes button. Changes made here will be saved as User default Settings. This can be set as System Default settings by clicking Yes to the confirmation message "Display Settings updated. Do you want to set this as System Default?".

Class Selection

Class selection option provided under this icon is for flitering the Furniture Classes. Furniture Classes of all the Furniture will be listed with check boxes. Check the required Furniture classes to be displayed and click Save button. The Furniture with the selected Furniture Classes will be shown .

Place

This option is to place an warehoused Furniture into a Drawing. Select the Warehoused Furniture from the data grid and click Place option from icon. The Drawing list will be displayed. Open the required drawing. The system will prompt to select a space to place the Furniture. Click on the exact location where the Furniture is to be placed.The Furniture will be placed there. Multiple Furniture can be placed in a single Place operation, if the drawing is the same. After placing each Furniture, the system will prompt for the location of the next Furniture till all the selected Furniture are placed.

Furniture having no Furniture Symbols cannot be placed in a drawing.

Warehouse

This option is to de-link the assigned Furniture from the drawing. Select the Furniture to be de-linked from the data grid and click Warehouse option from icon, a message prompting 'Are you sure you want to warehouse the selected Furniture?' will be displayed. On clicking Yes to the message, the selected Furniture will be de-linked from drawing.

Attachments

Users can add/edit/delete/view/download attachments to a furniture using this option. Select the required Furniture and click the Attachments icon. The files that can be attached will be restricted to data files like Word, Excel, pdf, jpg, gif, bmp etc.

Add: An Attachment can be added by clicking on the Add icon. The New Attachment window will be displayed. Select the Attachment Category first, and then select the required file. The attachment has to be uploaded by clicking on the Upload button.

Edit: An Attachment can be edited by clicking on the Edit icon.

Select an attachment and click on the Edit icon. The Edit Attachment window will be displayed with the current file name. The user can select the attachment category and the required file. Changes can be saved by clicking the Save Changes button.

Delete: Delete an attachment by selecting the required file and clicking the Delete icon.

View: The attachment can be viewed by selecting the file and then clicking the View icon. The selected attachment will be opened in an attachment viewer. View is supported only for image files.

Download: Attachment can be downloaded and saved in the user's computer by clicking the Download icon.

Show Drawing

This option is for showing the selected Furniture in the drawing. Select the Furniture from the Data Grid and click on the Show Drawing icon . The drawing will be displayed with the selected Furniture hatched and blinking.

Export

This option is to export the Furniture data to the Excel. On clicking Export icon, data will be exported to Excel.

Warranty Details

Users can add/edit/delete/ warranty details to the Furniture using Warranty Details option. Select the required Furniture and click the Warranty Details icon.

Add: Warranty details can be added by clicking the Add icon. New Warranty Details window will be displayed. Select a Warranty Type and Start Date, then enter Warranty Period In Months. User can also enter Warranty Certificate Number and Description if needed. The details can be saved by clicking on the Save button.

Edit: Warranty Details can be edited by selecting it and then clicking the Edit icon. The Edit Warranty Details window will be displayed with the saved details. The user can select Warranty Type and Start Date, then modify Warranty Period In Months. User can also enter Warranty Certificate Number and Description if needed. Changes can be saved by clicking the Save Changes button.

Delete: Delete Warranty Details by selecting it and then clicking the Delete icon.

Alert: This option is used to set Warranty Alert against each Warranty details. Select a Warranty Detail and click Alert icon, then Warranty Alert page will be displayed. User can select a Notification Template and then the Email recipients can be selected from iDrawings Users or Contacts menu. Alert can be saved by clicking Save Alert button.

Keyword search

This option helps users to search for Furniture details using keywords. The keyword could belong to any data field of the Furniture data. The user need not know or specify which field to look in. The Keyword could be any part of the Furniture data viz. Furniture No, Furniture Class Name etc. On entering the keyword and clicking on the Search icon the data grid will be filtered with Furniture records having data matching the specified keyword.

Advanced search

This option helps users to search for Furniture details with specific data in known data fields. The data being searched for can be selected from the list boxes or entered into text boxes provided against the data fields. The user can select one or more data fields viz. Furniture No, Furniture Class Name, Status etc or the additional data fields. After entering the search criteria and clicking Search button, the data grid will be filtered with furniture records which satisfy the selection criteria.

Fields to be displayed in the Search window can be set using the Configure icon, provided in the Search window. Search criteria entered in the Search window can be cleared by clicking the Clear button provided in the Search window. After clearing the Search criteria and click Search button to clear the Search Results in the Operational Data grid.
