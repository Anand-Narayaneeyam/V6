This option is used for creating, closing and reopening the Projects.

When clicking the Projects menu, the list of all projects will be displayed with Project Category drop down on the top of the list. Project Category dropdown is provided with General.

Add: Create a new project by clicking Add icon. General category will be displayed as selected and disabled in Project Category drop down. Enter the details like Project Name, Status, Project Number, Project Type, Project Manager, Description etc. in the form displayed. Project Type can be selected from the list of predefined Project Types and Project Manager can be selected from the list of users having access to the Projects Module. Click on Save button to add the new project.

Edit: This option is to edit an already added project. Click on Edit icon, the Edit project window will be displayed, select the Project to be edited, the details will be shown in the form. Make the necessary changes and then click the Save Changes button to save the changes.

Close: Close the project by selecting the project to be closed from the list and then clicking the Close icon.
