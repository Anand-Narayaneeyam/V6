The Gross Area will be the total Gross Area of all floors in all buildings in the site. Any additional data fields added for sites will also be displayed. Note: User can go to the Buildings list of a particular Site by clicking the hyper link provided at the Buildings count column of the required Site.

Edit: Edit a site by selecting it and clicking the Edit icon. Enter the Site Name and any additional data fields defined. By default the site is created with Active status. Only active sites will be available to users based on their access permissions.

Delete: Delete a Site by selecting it and clicking the Delete icon. It is not possible to delete a site having one or more Buildings added to it.

Close: Close a site by selecting the site from the list and then clicking the Close icon. If a site is closed, it will not be possible to work with the site again until it is reopened.

Reopen: Reopen a closed site by selecting it and then clicking the Reopen icon.

Attachments

Administrators can add/edit/delete/view/download attachments to a site using this option. Select the required site and click the Attachments icon. The files that can be attached will be restricted to file types like Word, Excel, pdf, jpg, gif, bmp etc. On selecting Attachments icon, the File attachments window opens up showing any files that are already attached to this site. Attachments can now be added to the selected site.

Add: An attachment can be added by selecting the Attachment Category first, and then selecting the file through the file open dialog by clicking on the Browse button. The attachment has to be uploaded by clicking on the Upload button.

Select an attachment and click Edit icon. The Edit Attachment window will be displayed with the current file name. The user can select the Attachment Category. There is an option to replace the selected attachment with a new file. (It is possible to browse the file location by clicking the Browse button).

Delete: Delete an Attachment by selecting it and clicking the Delete icon

View: The attachment can be viewed by selecting the file and then clicking the View icon.

Download: Attachment can be downloaded and saved in the user's computer by clicking the Download icon.

Keyword Search

This option helps users to search for Building details using keywords. The keyword could belong to any data field of the Building data. The user need not know or specify which field to look in. The Keyword could be any part of the Building data viz. Name, Code, Status, Floor Count, Area, etc. or the additional data fields.

On entering the keyword and clicking on the Search icon, the data grid will be opened with Building records having data matching the specified keyword.

This option helps users to search for Site details with specific data in known data fields. The data being searched for can be selected from the list boxes or entered into text boxes provided against the data fields. The user can select one or more data fields viz. Name, Code, Status, Time Zone, Area, etc. or the additional data fields. After entering the search criteria and click Search button, the data of the sites which satisfy the selection criteria. When a search is done, the search criteria will be saved and available for subsequent searches in the current session until another search is done with new criteria.

Fields to be displayed in the Search window can be set using the Configure icon, provided in the Search window.

Search criteria entered in the Search window can be cleared by clicking the Clear button provided in the Search window. After clearing the Search criteria and click Search button to clear the Search Results in the Site Data grid.
