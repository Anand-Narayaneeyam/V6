Trades are the job skills or specialization areas like Electrician, Plumber, Carpenter, etc. that required for performing a task.

Administrators can add trades, with trade code and can set the hourly rate for the trade.

Add: New Trade can be added by clicking the Add icon.

Edit: Edit a Trade by selecting it and then clicking the Edit icon.

Delete: Delete a Trade by selecting it and then clicking the Delete icon.

Technicians are generally in-house staff members who perform work orders. Each Technician would be identified with a Trade.

Administrators can add new technicians, with technician code, and specify their trades.

Add: New technician can be added by clicking the Add icon..

Edit: Edit a technician by selecting it and then clicking the Edit icon.

Delete: Delete a technician by selecting it and then clicking the Delete icon.

Contractors are external individuals or organizations who perform work orders. A contractor would be associated with a Trade. Administrators can add new contractors for different trades.

Add: New contractor can be added by clicking the Add icon.

Edit: Edit a contractor by selecting the row and then clicking the Edit icon.

Delete: Delete a contractor by selecting the row and then clicking the Delete icon.

Information about Manufacturers of Assets like Contact name, title, address, phone, email, etc. can be stored in the system. This information can be used for taking action or giving feedback based on the performance/lifetime analysis of equipment supplied by them.

Add: New manufacturer can be added by clicking the Add icon.

Edit: Edit a manufacturer by selecting it and then clicking the Edit icon.

Delete: Delete a manufacturer by selecting the row and then clicking the Delete icon.

The parts and spares required for performing work orders can be defined through this option. The cost per unit and manufacturer name can also be entered.

On selecting this option, all parts will be displayed with details like manufacturer name, cost per unit, etc.

User can associate Equipment Classes to the selected Part using Associated Equipment Classes icon.

Add: New Part can be added by clicking the Add icon.

Edit: Edit a part by selecting it and then clicking the Edit icon.

Delete: Delete a part by selecting it and then clicking the Delete icon.

Associated Equipment Classes:

This option is used to associate multiple Equipment Classes to the selected Part. Equipment Classes can be associated to the Part through Add icon. Equipment Classes associated with the Part can be removed using the Delete icon.

Add: Equipment Classes can be associated to the Part by clicking Add icon.

On clicking Add, New Equipment Class tab will be displayed with an option to select the Equipment Category. On selecting the required Equipment Category, all Equipment Classes (having Equipment) of the selected Category will be listed. Already added Equipment Classes will not be listed. User can select the required Equipment Classes and click Save button. All the selected Equipment Classes will be added to the Associated Equipment Classes list.

Delete: Equipment Classes added to the selected Part can be removed through this option. Equipment Class can be deleted by selecting the required Class and then clicking Delete icon displayed on selecting the required Card.

The tools required for performing work orders can be defined through this option with their hourly rates. On selecting this option all tools will be displayed with their hourly rates.

Add: New tool can be added by clicking the Add icon.

Edit: Edit a Tool by selecting it and then clicking the Edit icon.

Delete: Delete a tool by selecting it and then clicking the Delete icon.

Different levels of Priority for Work Requests like High/Medium/Low can be defined through this option.

Add: New Priority can be added by clicking the Add icon.

Edit: Edit a Priority by selecting it and then clicking the Edit icon.

Delete: Delete a Priority by selecting it and then clicking Delete icon.

Work Orders can be put on hold for different reasons like Waiting for Building Engineer, Waiting for gathering information from Requester, Waiting for Government Approval, Waiting for Materials, Waiting for Other Reasons, Waiting for Outside Contractor etc. The reasons for hold can be defined through this option.

Add: New Reason can be added by clicking the Add icon. A new Card will be inserted on clicking Add icon. Enter the Reason and Description and then click the Save (tick) button.

Edit: Reason can be edited by clicking the particular field of the required reason twice or by clicking the Edit icon displayed on the card, when mouse points to the card.

Delete: Reason can be deleted by clicking the Delete icon displayed when a card is selected.
