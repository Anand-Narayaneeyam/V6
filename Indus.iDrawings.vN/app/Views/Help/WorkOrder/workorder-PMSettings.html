A PM Procedure defines the job steps to be performed and the safety steps to be followed while performing a PM procedure/activity for an Equipment Class. This can be used as a checklist while performing the Work Order. Job Steps and Safety Steps can be added to a procedure from Job Steps / Safety Steps icons from Procedure. Multiple Equipment Classes may use same procedure. These can be specified as Associated Equipment Classes of the Procedure. Attachments can be added to Procedures through Attachments icon.

Add: New PM Procedures can be added by clicking the Add icon. After adding a Procedure, message "Do you want to associate Equipment Classes to this Procedure?" with Yes and No buttons will be displayed. By selecting Yes, User can associate Equipment Classes to the selected Procedure using Associated Equipment Classes icon.

Edit: Edit a PM Procedure by selecting it and then clicking the Edit icon.

Delete: Delete a PM Procedure by selecting it and then clicking the Delete icon. A Procedure cannot be deleted if it is used anywhere in the PM module.

Associated Equipment Classes: Multiple Equipment Classes can be added to a PM Procedure by selecting it and then clicking the Associated Equipment Classes icon.

This option is used to associate multiple Equipment Classes to the selected Procedure. Equipment Classes can be associated to the Procedure through Add icon. Equipment Classes associated with the Procedure can be removed using the Delete icon.

Add: Equipment Classes can be associated to the Procedure by clicking Add icon.

On clicking Add , New Equipment Class tab will be displayed with an option to select the Equipment Category. On selecting the required Equipment Category, all Equipment Classes (having Equipment) of the selected Category will be listed. Already added Equipment Classes will not be listed. User can select the required Equipment Classes and click on Save button. All the selected Equipment Classes will be added to the Associated Equipment Classes list.

Delete: Equipment Classes added to the selected Procedure can be removed through this option. Equipment Class can be deleted by selecting the required Class and then clicking Delete icon.

Job Steps: Job Steps can be added to/deleted from a PM Procedure by selecting it and then clicking the Job Steps icon.

Job Steps are the different jobs to be performed in a step by step manner while performing a PM activity. Each Step has to be entered separately through Add icon. A new job step can be defined with Description of the Job Step.

A Job Step, already created can be modified by clicking Edit icon and it can be removed from the procedure by selecting Delete icon.

Add: New Job Step can be added to the Procedure by clicking Add icon.

Edit: Edit a Job Step by selecting it and then clicking the Edit icon.

Delete: Delete a Job Step by selecting it and then clicking the Delete icon.

Safety Steps: Safety Steps can be added to/deleted from a PM Procedure by selecting it and then clicking the Safety Steps icon.

Safety Steps are the different Safety Steps to be performed in a step by step manner while performing a PM activity. Each Step has to be entered separately through Add icon. A new Safety step can be defined with Description of the Safety Step.

A Safety Step, already created can be modified by clicking Edit icon and it can be removed from the procedure by selecting Delete icon.

Add: New Safety Step can be added to the Procedure by clicking Add icon.

Edit: Edit a Safety Step by selecting it and then clicking the Edit icon.

Delete: Delete a Safety Step by selecting it and then clicking the Delete icon

Attachments: This option is to add/edit/delete/view/download attachments to/from Procedures. Select a Procedure and click Attachments icon; an attachment window will be displayed with options to add/edit/delete/view/download attachments.

The File attachments window opens up on selecting Attachments icon, showing any files that are already attached to this site. Attachments can now be added to the selected Procedure.

Download: Attachment can be downloaded and saved in the user's computer by clicking the Download icon.

Master PM Schedules are templates for PM Schedules. PM General / Routes can make use of the Master PM Schedules while creating the schedules. On creating PM Schedules with Master PM, all the scheduling information (Equipment Category, Class, Recurrence Pattern or Meter Field details, Procedures etc) of the selected Master PM Schedule will be copied automatically to the new PM Schedule created.

Master PM Schedule is not used for generating Work Orders as in other PM Schedules; instead it is used as a template for creating PM Schedules. By modifying Master PM Schedule, all the associated PM schedules can be changed.

Master PM Schedule can be created as Time based PM Schedules. Time based Master PM Schedules are generally used for creating time based PM Schedules (General or Route), which can be used for generating Work Orders in scheduled Intervals (Daily, Weekly, Monthly or Yearly) using different Recurrence Patterns within each interval (Example: The First Monday of every month).

Time based Master PM Schedules can be created by selecting Equipment Category, Equipment Class, Perform on, Recurrence Pattern and Seasons. Perform on combo box will be populated with 'Any Day', 'Working Day' and 'Nonworking day' options. By default Perform on combo box will be displayed with 'Any Day' option and Work Orders can be generated for any day regardless of the selected Calendar. If the 'Working Day' option is selected in the Perform on combo, Work Orders are generated only for Working Days else if Nonworking Day option is selected, Work Orders are generated only for Nonworking Days.

Recurrence Pattern in the Add/Edit Schedule is provided with options - Daily, Weekly, Monthly and Yearly. 'Daily' option will be selected by default. As the name indicates, Daily option is for generating the PM Work Orders every day, Weekly option is for generating PM Work Orders weekly, Monthly option is for generating PM Work Orders monthly and Yearly option for generating PM Work Orders yearly.

When selecting the Weekly option, all the Week days will be displayed with checkboxes. User can select any weekdays by checking its corresponding check boxes. For example, if PM Work Orders have to be generated on every Mondays and Wednesdays, User has to select Monday and Wednesday from the Weekdays listed.

On selecting the Monthly/Yearly option, the corresponding details of the Monthly/Yearly option will be listed in which the user can select a particular day in a month/year. By default 'None' option will be selected for Monthly/Yearly option. For Example – if Monthly is selected, User can specify the particular day as Day 21 of a month or First Saturday in a Month. If Yearly is selected, the required day can be specified by providing 2nd March of every Year, or First Saturday in April etc.

For Time based Schedules, any number of Procedures can be added.

A new Master PM Schedule can be created using Add icon. Also an already existing Schedule can be modified using Edit icon. Using Delete icon, a Master PM Schedule added can be deleted. All Procedures associated with the Master PM schedule will be listed on clicking Procedures icon.

Add: New Master PM Schedules can be added by clicking the Add icon. Master PM Schedule Name, Equipment Category, Class, Status and Interval (Frequency) need to be specified on adding a new Master PM Schedule. After adding a Master Pm Schedule, a message "Do you want to add Procedures to this Schedule?" with Yes and No buttons will be displayed. It will be directed to the Procedure Add page when Yes button is clicked for the confirmation message.

Edit: Edit a Master PM Schedule by selecting it and then clicking the Edit icon.

Delete: Delete a Master PM Schedule by selecting it and then clicking the Delete icon. If a Master PM Schedule is used in PM Schedule, it cannot be deleted.

An Elevator has 3 procedures namely Monthly Procedure, Quarterly Procedure and Semi Annual Procedure to be executed. This can be done by creating a Monthly Schedule and by adding these procedures with sequence number as 1, 3 and 6 for Monthly, Quarterly and Semi Annual Procedures respectively. (If more than one procedure falls due on the same date, only the procedure with the higher sequence number will be selected. If 1 is also to be done with 3, the job steps of 1 should be included in 3).

PM Schedules are predefined schedules for doing the scheduled preventive maintenance work. They are used to generate preventive maintenance work orders. A PM Schedule is defined for specific Equipment belonging to an Equipment Class. It specifies the PM procedure to be performed, the starting date and the frequency (Interval) with which it should be repeated. Seasons can also be specified for the Schedules.

PM Schedules can be classified as PM Schedule (General) and PM Schedule (Routes). As the name indicates PM Schedule (Routes) is for Equipment grouped under a Route and PM Schedule (General) for any Equipment.

PM Schedule can be created as Time based PM Schedules. Time based PM Schedules are generally used for creating PM Schedules based on time (General or Route), which can be used for generating Work Orders in scheduled Intervals (Daily, Weekly, Monthly or Yearly) using different Recurrence Patterns within each interval (e.g. The First Monday of every month).

Time based PM Schedules can be created by selecting Equipment Category, Equipment Class, Perform on and Recurrence Pattern. Perform on combo box will be populated with 'Any Day', 'Working Day' and 'Nonworking day' options. By default Perform on combo box will be displayed with 'Any Day' option and Work Orders can be generated for any day regardless of the selected Calendar. If the 'Working Day' option is selected in the Perform on combo, Work Orders are generated only for Working Days else if Nonworking Day option is selected, Work Orders are generated only for Nonworking Days.

Recurrence Pattern in the Add/Edit Schedule is provided with options - Daily, Weekly, Monthly and Yearly. 'Daily' option will be selected by default. As the name indicates, Daily option is for generating the PM Work Orders every day, Weekly option is for generating PM Work Orders weekly, Monthly option is for generating PM Work Orders monthly and Yearly option for generating PM Work Orders yearly.

When selecting the Weekly option, all the Week days will be displayed with checkboxes. User can select any weekdays by checking its corresponding check boxes. For example, if PM Work Order has to be generated every Monday, User has to select Monday from the Weekdays listed. If PM has to be done two days a week, say every Monday and Thursday, then both days can be checked.

On selecting the Monthly/Yearly option, the corresponding details of the Monthly/Yearly option will be listed in which the user can select a particular day in a month/year. By default 'None' option will be selected for Monthly/Yearly option. For example, if Monthly is selected, User can specify the particular day as Day 21 of a month or First Saturday in a Month. If Yearly is selected, the required day can be specified by providing 2nd March of every Year, or First Saturday in April etc.

Date in Service field is used to calculate the Next PM Date based on the options provided in the Recurrence Pattern. Number of PM Work Orders generated with a schedule can be limited by selecting the required number in the 'End after' field or by selecting a particular Date from the 'End by' field in the Range of Recurrences.
Examples –

For a Weekly PM Schedule, if the Date in Service is 12 Jun 2014 (Thursday) and Saturday is selected for PM, Weekly PMs will be scheduled on all Saturdays after one week from the Date in Service, i.e. starting from 21 Jun 2014. No PM will be scheduled for the first Saturday following the Date in Service, i.e. 14 June 2014 as one week has not elapsed from the Date in Service.

For a Monthly PM Schedule, if the Date in Service is 12 Jun 2014, and 'None' option is selected, Monthly PMs will be scheduled for the 12th of every month, starting from July. If a particular day is selected instead of None, say Day 16, the next PM date will be the 16th of the next month, provided one month has elapsed from the Date in Service. If not, it will be pushed to the 16th of the following month. If a month does not have the specified day, say 29, 30 or 31 in February, or 31 in April, June, September and November, it will be scheduled for the next day (or last day of the month).

For a Yearly PM Schedule, if the Date in Service is 12 Jun 2014, and 'None' option is selected, Yearly PMs will be scheduled for the 12th June of every year, starting from 2015. If a particular day is selected instead of None, say Day 16 of June, the next PM date will be the 16th June of the next year, provided one year has elapsed from the Date in Service. If not, it will be pushed to the specified day of the following year. If a month does not have the specified day, say 29, 30 or 31 in February, or 31 in April, June, September and November, it will be scheduled for the next day (or last day of the month).

Apply Updates from Master PM - Master PM Schedule can be specified for the PM Schedule. Master PM drop down will be listed with all the Master PM based on the access privilege of the Logged in User. On selecting a Master PM, all the scheduling information (Equipment Category, Class, Interval (frequency) or Meter field details etc.) of the selected Master PM Schedule will be automatically copied to the new PM Schedule created. If the 'Apply Updates from Master PM' check box is selected and the Schedule is created, all the changes made later in Master PM Schedules will be automatically applied to all PM Schedules which uses the Master PM Schedule Template.

Seasons – Seasons Field list all the Seasons added to the Customer with checkboxes. If any Season is checked, the PM Work Orders will be generated only in the selected Season (only if Next PM Date falls within the Season Start and End Date).

Auto generate Work Order? - An Option to generate the PM Work Order automatically is provided with the PM Schedule. Work Order can be generated automatically before the specified number of days of the next PM date by enabling the ‘Auto Generate Work Order?’ check box. On selecting the checkbox, the list boxes for specifying the number of days before which the Work Order has to be generated and the Work Type through which the PM Work Orders to be reviewed further will be enabled. Action Point User list box will be enabled on selecting Work Type. Enter the number of days before which the Work Order has to be generated automatically and select the Work Type. If the Work Type selected has the first Action Point of Specific User Type, the Action Point user also has to be selected from the User combo. PM Work Order(s) according to the selection is auto generated and will be available for further review for the corresponding user(s) in Work Order module under Current Tasks: PM Work Orders menu.
