Work Order reports are available under Reports icon.

The reports generated by the system can be viewed using the controls available in the report viewer. The report can be saved to Excel, PDF or RTF format by clicking on Excel, PDF or RTF button.

The following reports are available in Work Order module.

General - Equipment Work History

This report shows the work history of the selected Equipment. On selecting the report, a Search Equipment page will be displayed with the options to select Equipment Category and Class. Select the Equipment Category and Equipment Class from the drop down list. All the Equipments having Requests/Work Orders in Completed or Closed Status will be listed. Select the required Equipment and click Next. Report will be generated with details of equipment work history based on Service Requests, Non PM and PM Work Orders if any of the selected equipment. Time Spent to complete field is provided in the Report, to get the down time of the selected Equipment. Only completed and closed Requests or work orders will be included in this report.

User can filter the report based on Entity Category namely Service Requests and PM Work Orders.

Service Requests - Workflow History

This report shows the status of Requests or Work Orders and history of the request up to the current stage.

Service Requests - Completed Workflow History

This report shows the history details of Requests or Work Orders which are in Completed status. (Completed but closed are not displayed here)

PM Workflow History

This report shows the status and history of PM Work Orders up to the current stage.

PM Completed Workflow History

This report shows the history details of PM Work Orders which are in Completed status. (Completed but closed are not displayed here.)

Monthly PM Summary Report

This report shows a summary of the PM Work Order Generated or Completed in the selected Date Range. It also shows the count of Work Orders completed before the Scheduled Date, on Scheduled Date and after the Scheduled Date.

On selecting this menu, a Select Work Type page followed by Select criteria page will be displayed with options to select the Equipment Category, Class, Number and Scheduled Date Range (From and To Dates). Select the required criteria and click Generate button. A Monthly PM Summary Report will be displayed. A drop down to select the Recurrence Pattern is provided in the Report and by default 'All' is selected in the Recurrence drop down. User can filter the summary report based on the recurrence pattern listed in the drop down. A Detailed Report of Completed Work Orders can be viewed by clicking Details button, provided in the Report.
