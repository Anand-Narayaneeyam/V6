Generated PM WOs menu will be available under Work Order module icon.

PM Work Orders are generated based on the PM schedules created for Equipment. Work Orders have to be created for one Equipment Category/Class or one Route at a time, for the desired number of days.

User can choose a Calendar view for generating the Work Orders, by selecting the Calendar View checkbox

If Calendar view checkbox is not selected in the Select Criteria page, on clicking Next after the required selections (Equipment Category, Class or Route and Number of days for which work orders have to be created and Work Type) a Pending Schedules list based on the selected criteria will be listed with Generate icon.

Select the required Schedule(s), and click Generate icon. PM Work Orders corresponding to the selected Equipment Class/Route will be generated. Single or multiple Work Orders will be generated according to the next PM date of the selected Equipment Class/Route.

If Calendar view checkbox is selected in Select Criteria Page, on clicking Next button after the required selections (Route, Equipment Category and Class for the Pending Schedules to be viewed and Work Type) a Calendar view of current month will be displayed with Dates of Pending Schedules marked as Red and number of pending Schedules. The previous and next month calendar can be seen by clicking the Previous and Next icons provided.

Pending Schedules will be listed with details like PM, Procedures, Scheduled Date, Equipment Category, Class and Number and Next PM Date when clicking the Red color box. Work Orders can be generated by selecting the required Schedules and then by clicking the Generate icon.

When Generate icon is clicked, options to select the Notification Template and Email Recipients will be displayed. If the Work Type selected has the first Action Point is of Specific User Type, the Action Point user also has to be selected from the User combo.Email Recipients can be selected by required User Category section (iDrawings Users, Technicians or Contractors) after selecting the Notification Template.

On clicking the Generate button, PM Work Order(s) will be generated and will be available for further review for the corresponding user(s) in Work Order module under Current Tasks: PM Work Orders.

Work Order generated Notification Mails are sent to the action point users on generating the Work Orders. When generating the Work Orders manually with multiple Schedules selected, a notification mail, with the details of all PM Schedules combined will be sent to Action Point Users.

In the case of Auto generation of Work Orders, a Notification mail is sent to the first action point user with the details of all PM Schedules combined.

On clicking this menu, a Calendar view of current month will be displayed. The Green color indicates dates having schedules for which Work Orders already generated. The number shown in the color box shows the count of Work Orders generated. The previous and next month calendar can be viewed by clicking the Previous Month and Next Month buttons provided. Current month’s Calendar can be viewed by clicking the This Month button.

The detailed list of Generated Work Orders will be displayed by clicking the color box, with details like WO Number, PM, Procedures, Scheduled Date, Equipment Category, Class and Number, Status, Action by etc. Work Orders listed can be reviewed by clicking Review button.
