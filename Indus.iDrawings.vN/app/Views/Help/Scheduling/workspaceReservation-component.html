This option is used to book Hoteling Workspaces. Through this option it is possible to check the availability of seats / Workspaces. This option will be available only if Workspace Booking option is enabled for the customer.

Reserve Workspace and Reservations are provided under the Workspace Reservations menu. Two tabs namely My Reservations and Active Reservations are provided in the Reservations menu.

Reservations can be submitted for reserving the required Hoteling Workspaces. Reserve Workspace option helps users to book hoteling Workspaces.

My Reservations option is for processing Reservations created for the logged in User. User can modify or cancel the Reservations through My Reservations.

Active Reservations tab will be available only for Administrators and Concierges (Division Administrators). Active Reservations tab list all the Reservations made for the logged in Users Site and Division/Team and also list all the Reservations made by the logged in User. User can modify or cancel the Reservations in the Active Reservations.

Reserve Workspace

The user can reserve hoteling Workspace through Workspace Reservation: Reserve Workspace option. On selecting this option, all the Workspaces available for Reservation for the current date will be displayed by default with status as Available. Workspaces available for Reservation are those seats which are defined as Hoteling Seats.

Users can search for available hoteling Workspaces for a selected period. After entering the required Date Range, select the checkbox 'Show available Seats only' and click on Search button. Then all the available Workspaces will be displayed below.

A Filter link is provided to filter the workspaces based on Site, Building, Floor, Room Number and Space Function.

For a Concierge, radio buttons to choose Normal Workspaces and Special Use Rooms are provided. On selecting Normal radio button, all the Hoteling workspaces will be listed whereas on selecting Special Use Room radio button, all workspaces in the Room with Space Assignment Type – Special Use Room will be listed.

The details of a room can be viewed by clicking the Room No. link displayed in the Reserve Workspace page. On clicking this link, Room Details window will be displayed with all the details of the selected room. Any photo added for the room will also be displayed. A Day view calendar with time slot is provided in the Room Details. A User can book the required Workspace from the Room Details link, by clicking the required time slot.

The Reservation for the required workspace can be done by clicking the Book link displayed against each row of the Workspace. On selecting the link, Reserve Workspace page will be displayed where the user can select/enter Date and Time, Booked for and Comments details. A Check Availability button is provided in the Reserve Workspace window to check the availability of the selected Workspace in the selected Timeslot.

Booked for field will be displayed only if the logged in User is a Division Administrator (Concierge). All the Users belong to the logged in User’s Team / Division will be listed in the Booked for drop down. Search textbox is provided to search the required User in the Booked for field.

Once all the requirements are entered, the Reservation can be submitted by clicking the Reserve button. The newly submitted Reservation will be listed in the My Reservations menu of the Booked for User.

This option list the Workspace Reservations created for the Customer. Two tabs namely My Reservations and Active Reservations are provided in Reservations menu.

My Reservations

This option is to list all the Workspace Reservations booked for the logged in user. Through this option the user can modify, or cancel the reservations. Modification is possible for the reservations which are made in the accessible site.

All the active Reservations will be listed with 'Confirmed' status. A checkbox “Show active reservations” is provided in the My Reservations tab to show only Reservations in 'Confirmed' or 'Checked In' status. By default, the checkbox will be displayed as selected.

Reservation will be moved to 'Auto Bump' status if the Reservation time is lapsed and the user has not checked in the Reservation. If the User has checked in the Reservation from Hoteling App or KIOSK, and Reservation time is lapsed, the Reservation will be moved to 'Expired' status.

Modify Reservation: This option is to modify a Workspace Reservation. Modifications can be done for the Reservationscreated by the logged in User or reservations made in the accessible site for the logged in user. After modifying the Reservation, Workspace Reservation Modified Mail will be sent to the Requester and Booked for User

Note: Reservations in Confirmed status only can be modified.

Cancel Reservation: A Reservation can be cancelled by clicking the Cancel Reservation icon. User can cancel the Reservations in Confirmed status only. While cancelling the Reservation, status of the selected Reservation will be changed to Cancelled and Workspace Reservation Cancelled mail will be sent to the User selected in 'Booked for' and to the Requester.

Active Reservations

This option is used for viewing all the Workspace Reservations submitted by different users. This tab will be available only for Administrators and Division Administrators (Concierges). For an Administrator, all the Reservations made by different Users will be listed. All the Reservations made by the concierges or Reservations made in the accessible Site and Team will be listed for a Concierge.

A checkbox "Show active reservations" is provided in the Active Reservations tab to show only Reservations in 'Confirmed' or 'Checked In' status. By default, the checkbox will be displayed as selected.

Options to Modify and Cancel Reservations are provided in the Active Reservations tab.

By default, the Reservations will be listed in the ascending order of 'From' and 'To' dates of the Reservations. The list can be sorted on any column by clicking the column header.

Modify Reservation: This option is to modify a Workspace Reservation. Modifications can be done for the Reservations created by the logged in User or any Reservations made in the accessible site. After modifying the Reservation, Workspace Reservation Modified Mail will be sent to the Booked for User.

Note: Reservations in Confirmed status only can be modified.

Cancel Reservation: A Reservation can be cancelled by clicking the Cancel Reservation icon. User can cancel the Reservations in Confirmed status only. While cancelling the Reservation, status of the selected Reservation will be changed to Cancelledand Workspace Reservation Cancelled mail will be sent to the User selected in 'Booked for' and to the Requester.

Check In and Check Out can be done by the Booked for Users and this option will be available in Mobile or KIOSK application.
