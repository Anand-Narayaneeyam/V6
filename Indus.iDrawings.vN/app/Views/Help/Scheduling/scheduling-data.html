This section provides an option for the user to view and analyze the space data graphically.

Click on this, and the option for selecting the floor will be listed. Select the desired floors for which the data is to be viewed and click on the Show Data icon. The details will be listed in the Space data grid

Edit

Using this option, the administrator can edit the space details. Select the space to be edited and click on the Edit icon. The details will be displayed in a new window. Make necessary changes and click the Save Changes button.

The Room Number field will be visible only if it is enabled for the customer and it can be set as Unique for the Organization/Site/Building/Floor. It is also possible to have the Site or Building Code or both as prefix to the Room Number. This is disabled by default, but can be turned on for the customer.

If the Room Number is set to be unique and the entered room number already exists, it will display a message “Room No. already exists for the Organization/Site/Building/Floor” corresponding to the set Uniqueness. If any prefix has been set for Room Number for the customer, then a read-only prefix will be displayed before the Room Number Text box. Only the Room Number need be entered, the prefix will be automatically inserted.

By default Space Category is enabled. If the value selected is one of the Common Area space categories, like Building Common, Floor Common or Vertical Space it will not be possible to edit/set the Organizational Unit for this space, and all Organizational level combo boxes will be disabled. But if it is required to assign Organizational Units toCommon Spacesalso, Organizational Unit Assignment to Common Area can be enabled for the customer.

Data for additional data fields can also be entered through this data entry window.

Share Space

Provision for Sharing a Space between Multiple Organization Units is allowed. This is needed for sharing spaces like Consulting Rooms, Conference Halls, etc. between different departments.

If ‘Space Allocation to Multiple Departments’ option is enabled for the customer, a link ‘Share Space’ will be displayed along with Organizational Unit in the Data Entry window. The Administrator can view/add/edit/delete the sharing of space using this option. The Sharing among different Organizational Units should sum up 100.

If the selected Space is already shared, it will be indicated with a caption ‘Shared’ next to the Organizational Unit name. Click ‘View’ link to view the existing sharing details. Click Viewlink, the Space Sharing by Organizational Units window opens up with the percentages of already assigned Organizational Units.

Add: On clicking Add icon, Assign Shared Organizational Unit window will be displayed with Org. Units combo boxes and Percentage textbox. Select the required Organization Unit name from the required level and enter the percentage. The sharing details will be saved by clicking Add button.

Edit: Space sharing can be edited by clicking the Edit icon. The Sharing among different Organizational Units should sum up to 100.

Delete: Remove an assigned Organizational Unit by selecting the same and clicking the Delete icon.

Multiple Edit

Multiple records can be edited by selecting the required spaces using the Shift or Ctrl keys, and then clicking on the Edit icon. A window will be displayed listing the fields that can be edited. Select the required Field and enter or select the value to be updated to the selected Spaces and click Save Changes button. The changes have to be made selecting one field at a time.

Assign Space Standard

This option is for assigning space standard and space assignment type to spaces.Select Assign Space Standard icon.Space standard and Space Assignment Type can be assigned/de-assigned by invoking this option and then selecting the space for which space standard is to be assigned. Space Assignment Type can be set to Assignable, Scheduling Space and Unassignable. Scheduling Space option will be available only if Scheduling module is enabled for the customer.

If “Approval Process: Not required” in Room Booking or Seat Booking is enabled for the customer, the request will be confirmed directly without any approval process.

If Space Assignment Type is selected as Scheduling Space, an option to select the Amenities of the room and its count will be displayed. A room can be booked only if the Space Assignment type is assigned as Scheduling Space.

Note: If any Room booking requests exist for the selected Space, then the Space Assignment Type option in Assign Space standard page cannot be edited and will be displayed as read only.

Totalize

This option is to show the Total Count, Gross Area Total & Net Area Total of the selected spaces.

Select the records to be totalized and click on Totalize icon. A message box will be displayed with the Number of Spaces selected, and the Gross and Net Areas of the selected spaces.

Apart from this, if any additional data fields of Space are set as Totalizable, the total value of such fields also will be shown.

Attachments

Users can add/edit/delete/view/download attachments to a Space using this option. Select the required Asset and click the Attachments icon. The files that can be attached will be restricted to data files like Word, Excel, pdf, jpg, gif, bmp etc.

Add – An attachment can be added by selecting the Attachment Category, and the file. The attachment has to be uploaded by clicking on the Save button.

Edit – An Attachment can be edited by clicking on the Edit icon The Edit Attachment window will be displayed with the current file name. The user can select the attachment category and the required file open. The attachment has to be updated by clicking on theSave Changes button

Delete - Delete an attachment by selecting the required file and clicking the Delete icon.

View – The attachment can be viewed by selecting the file and then clicking the View icon.The selected attachment will be opened in an attachment viewer. View is supported only for image files

Download – Attachment can be downloaded and saved in the user’s computer by clicking the Download icon

Show Drawing

This option is for showing the selected Space in the drawing. Select the Space record from the Data Grid and click on the Show Drawing . The drawing will be displayed with the selected Space hatched.

Show Zoomed

This option is for showing the selected Space in drawing in the zoomed in position. Select the Space record and click on the Show Zoomed icon. The drawing will be displayed in the zoomed in position with the selected space hatched.

Export

This option is to export the space to the Excel. On clicking Export icon, a confirmation message, “Do you want to export the data along with Space Handle?” with Yes and No buttons will be displayed. On clicking Yes to the message, data will be exported with space handle. Else it will be exported to excel without space handle.

If any Shared Spaces exist, such spaces will be exported as separate rows for each organizational unit in the shared space. All space details except the area details and organizational units will be same for those rows.

Keyword Search

This option helps users to search for space details using keywords. The keyword could belong to any data field of the space data. The user need not know or specify which field to look in. The Keyword could be any part of the space data viz. space category, space function, room number, cost category, area, etc. or the additional data fields.

On entering the keyword and clicking on the Search icon, the data grid will be filtered with space records having data matching the specified keyword.

A facility user would be able to view only those floors/divisions for which privileges are granted to him.

Advanced Search

This option helps users to search for space details with specific data in known data fields. The data being searched for can be selected from the list boxes or entered into text boxes provided against the data fields. The user can select one or more data fields viz. space category, space function, room number, cost category, area, etc. or the additional data fields. After entering the search criteria and click Search button, the data grid will be filtered with space records which satisfy the selection criteria.

Fields to be displayed in the Search window can be set using the Configure icon, provided in the Search window.

Search criteria entered in the Search window can be cleared by clicking the Clear button provided in the Search window. After clearing the Search criteria and click Search button to clear the Search Results in the Space Data grid.
