This option is available under Scheduling Module icon. When selecting the Scheduling icon, all the menu options for Scheduling Module will be displayed in the Left panel.

This section provides an option for the users to view the drawings enabled for Scheduling Management. List of all drawings will be displayed, organized by Site/Building/Floor, with the Drawing File Names and File Size. This list can be sorted on any column by clicking on the column header.

View Drawing

On selecting a drawing and clicking on the View icon, the drawing will be opened in a new window with a horizontal menu bar and data in the right side. The Space Data is fetched from the database and displayed in the drawing in dynamically created layers. Only the Space Polyline and Space Data layers will be turned on by default.

The data corresponding to the selected drawing also will be displayed on the right side of the Drawing. On bringing the mouse pointer over a space, a Tooltip will be displayed, showing the space data associated with the space.

The Right Click menu options provide an easier access to the frequently used functionalities of the Scheduling module for the users, in turn saving time and effort.

On right clicking the mouse inside a Space in the drawing, a pop-up menu will appear with the options Edit Space, Assign Space Standard, Select Multiple Spaces by Window, Pan, Zoom in, Zoom Out, Zoom Window, Zoom Extents and Layers

A brief description on the Right click options provided in Drawings of the Scheduling module is provided below.

Edit Space: This option is for editing the Space Data in the Drawings.

Assign Space Standard: This option is for assigning space standard and space assignment typeto any desired space. Space standard can be assigned/de-assigned by selecting the space, to which the space standards is to be assigned. Space Assignment Type can be set to Assignable, UnAssignable, and Scheduling Space. Scheduling Spaceoption will be available only if Scheduling option is enabled for the customer.

Select Multiple Spaces by window: This option is for selecting multiple spaces in the drawing by window

Pan: This option enables the user to pan the drawing. On selecting this option, the mouse pointer will change to a hand symbol. Now click the left mouse button, and without releasing it, move the mouse to pan the drawing.

Zoom In: This option enables the user to Zoom in the Drawing.

Zoom Out: This option enables the user to zoom out the Drawing.

Zoom Window: This option enables the user to zoom the Drawing to a Window selection.

Zoom Extents: This option enables the user to view the Drawing fully within the drawing display window by resetting any zoomed or panned states.

Layers: This option enables turning On and Off the layers in the base drawing. The dynamically generated layers are not available here. These have to be controlled through the Display Settings. Clicking on the Default button turns on the default space polyline layers.

The following menu options are available to work with the drawings.

Distribution Map

This option is available only after clicking the Distribution Map icon

This option allows users to get the distribution of space areas across different Organizational Units (Divisions, Departments, Cost Centers etc as defined in the Organizational structure) or based on the Validated Fields. The Spaces belonging to different Organizational Units or the Validated Fields are hatched in the drawing with different colors and hatch pattern. Users can select the Organizational Units or the validated fields for which the distribution is required.The color displayed according to the settings in Settings: Space: General Settings: Distribution Map Settings and can be changed by the user if required.

Space Tools

All the Space related operations such as Move Legend, Select Multiple Spaces by Window, Totalize, Measure Distance, Measure Area, Reserve a Room, Reserve a Seat and Move a Seat are grouped under Space Tools icon .

Edit Space

This option is to edit the Data of the selected Space. On selecting Edit Space option from Space Tools icon, user will be prompted to the select the space to be edited and the cursor turns to cross. Edit Space Data window will be displayed on clicking the required Space in the drawing. Make necessary changes and click the Save button. Space Data edit is possible only to users having Edit privilege.

By default, Space Category is enabled. If the value selected is one of the Common Area space categories, like Building Common, Floor Common or Vertical Space it will not be possible to edit/set the Organizational Unit for this space, and all Organizational level combo boxes will be disabled. But if it is required to assign Organizational Units to Common Spaces also, Organizational Unit Assignment to Common Area can be enabled for the customer.

Data for additional data fields can also be entered through this data entry window.

Share Space

Provision for Sharing a Space between Multiple Organization Units is allowed. This is needed for sharing spaces like Consulting Rooms, Conference Halls, etc. between different departments.

If ‘Space Allocation to Multiple Departments’ option is enabled for the customer, a link Share Space will be displayed along with Organizational Unit in the Data Entry window. The Administrator can view/add/edit/delete the sharing of space using this option. The percentage sharing among different Organizational Units should sum up to 100%.

If the selected Space is already shared, it will be indicated with a caption ‘Shared’ next to the Organizational Unit name. Click View link to view the existing sharing details. Click View link, the Space Sharing by Organizational Units window opens up with the percentages of already assigned Organizational Units.

Add: On clicking Add icon, Assign Shared Organizational Unit window will be displayed with Org. Units combo boxes and Percentage textbox. Select the required Organization Unit name from the required level and enter the percentage. The sharing details will be saved by clicking Add button.

Edit: Space sharing can be edited by clicking the Edit icon. The Sharing among different Organizational Units should sum up to 100.

Delete: Remove an assigned Organizational Unit by selecting the same and clicking the Delete icon.

Assign Space Standard

This option is for assigning space standard and space assignment type to any desired space. On selecting Assign Space Standard option from Space Tools icon, user will be prompted to the select the space for assigning the Space standard and the cursor turns to cross. Assign Space Standard window will be displayed on clicking the required space in the Drawing. Space Standard can be assigned/de-assigned to the selected Space. Space Assignment Type can be set to Assignable, Unassignable, and Scheduling Space. Scheduling Space option will be available only if Scheduling option is enabled for the customer.

By default “Approval Process: Not required” is selected in Room Booking or Seat Booking. So the Room / Seat booking request will be confirmed directly without any approval process.

If Space Assignment Type is selected as Scheduling Space, an option to select the Amenities of the room and its count will be displayed.A room can be booked only if the Space Assignment type is assigned as Scheduling Space.

If Seat Booking is enabled for the customer and the Space Assignment Type is selected as Assignable, a link named ‘Seats’ will be displayed against Room Seating Capacity option to define seats for the selected space. On selecting this link, Seats window will be displayed listing all the available seats (if any).

User can add or delete single/multiple seat(s) to/from the selected space using the following options listed in the Seats page. An option to set seats as Hoteling Seats also will be displayed in the Seats list page.

Add Single Seat: The user can add seats to the selected space, one at a time using this option. To add a single seat to the selected space, the user has to select Add icon and select ‘Single’ radio button. Then options to specify the Prefix and the Number of the specific seat will be displayed in the Add Seat window. The number of seats that can be added is limited to the Minimum Seating Capacity assigned to the selected space.

Add Multiple Seats: The user can add multiple seats at a time to the selected space using this option. To add multiple seats to the selected space, the user has to select Add icon and select ‘Multiple’ radio button. Then options specify the Prefix of the seats and the number range in which the seats are to be added in the Add Seat window. The number of seats that can be added is limited to the Minimum Seating Capacity assigned to the selected space.

Move Legend

This option is to move a legend from one point to another. Select Move Legend option from Space Tools icon , a toast message will be displayed, prompting 'Select an insertion point to add legend’. Then the mouse pointer turns to cross, and click on the location where the legend is to be moved.

Select Multiple Spaces by Window

This option is for selecting multiple spaces in the drawing by window, for viewing details of space data linked to those spaces.

Click on the Select Multiple Spaces by Window option from Space Tools icon , a toast message ‘Select a window, click to select/deselect space, Right Click to exit’ is shown. Then the mouse pointer becomes a cross. Select a window by clicking and dragging inside the drawing. The selected spaces will be hatched. Right Click when all required spaces are selected, the Data grid in the drawing page will be highlighted with the data of the selected spaces.

A selected space can be deselected if required, by clicking on the De-Hatch option in the Right Click or by clicking Tools: De-Hatch option.

Totalize

This option is to show the number of spaces selected, Gross Area Total & Net Area Total of the selected spaces.

Select the spaces to be totalized by Select Multiple Spaces by window and click Totalize option from Space Tools icon , a message box will be displayed with details like Number of Spaces selected, Gross Area Total, etc. of the selected spaces.

Apart from this, if any additional data fields of Space are set as Totalizable, the total value of such fields also will be shown.

The details shown in Totalize window can be added as legend by clicking the Add Legend button in the Totalize window.

Measure Distance – Point to Point

This option is used to measure the distance between two points. Select Measure Distance - Point to Point option from Space Tools icon and the cursor will turn into a cross. Click on the first point from which the distance is to be measured and then click on the end point, the distance between the two points will be displayed.

Select the option first, and the system will prompt to select the points to Measure Distance. Click the starting point from which the distance is to be measured, and then click the end point. The distance between the two points will be displayed. This marked distance can be removed by clicking the De-hatch option.

Measure Distance – Wall to Wall

This option is used to measure distance between two Walls. Select Measure Distance - Wall to Wall option from Space Tools icon and the cursor will turn into a cross. Click close to the wall from which the distance is to be measured. The starting point will snap to the nearest space polyline (Net Polyline for Net only Customers). Then click close to the other wall to which the measurement is to be taken. The end point will snap to the nearest point of the corresponding space polyline/net polyline. The distance between the two walls will be displayed.

Measure Area

This option is used to measure the area of a polygon in a Drawing. To measure an area, Select Measure Area option from menu and the cursor will turn to a cross. Mark the area to be measured by clicking on any one vertex point, and then clicking on the remaining vertex points one after the other. After the last vertex, close the polygon by right clicking in the drawing. The polygon will be hatched and the area will be displayed. This marked distance can be removed by clicking the De-hatch option.

Reserve a Room

New Room Scheduling Requests can be created directly from the drawing through this option. This option is available under Space Tools icon .

On selecting this option, all the scheduling spaces will be shown hatched and a message ‘Select the Space for Scheduling’ will be displayed. Select one of the hatched spaces for scheduling by clicking on it. A Month/Time view of the Calendar will be displayed with all the days of the selected Month and time as half hour slots. The time slots with booking will be highlighted in red color. Tool tip is provided for the highlighted slots to show the Event Names and their time slots.

New Room Booking scheduling request can be added by clicking on the required time slot from the calendar view with date and time.

New Room Booking Request page will be displayed with Reserve Room title where the User can enter/select From (Date time), To (Date time), Event Name, Participants (Count), Booked for,Invitees, Amenities, Services, Catering and Comments.

Check Availability button will be displayed to check the availability of the Room in the selected time slot.

On clicking Location button, the details of the Location which includes the Site, Time Zone, Building, Floor, Space Function and Room No. will be displayed as read only.

The requester can specify the number of participants and also select Invitees for the meeting.

The user can add Invitees, Amenities, Services and Catering as required while reserving the room. Invitees can be selected from iDrawings Users and/or Employees present in the Employee module.

Add Invitees: Invitees can be added by clicking the Add icon placed at the right side of the Invitees list box. On clicking the Add icon, Invitees page will be displayed to select the invitees. User has to select the required User Category from the combo and all the Users of selected User Category will be listed. iDrawings Users and Employees will be listed in the User Category combo. Select the required Invitees and click Save button. Selected Invitees will be added. Invitees can be removed from the list by selecting the Invitee to be removed and then clicking the '-' button.

Amenity/Service/Catering: Amenity/Service/Catering requirements needed for the meeting while reserving a room can be added byclicking the Add icon placed at the right side of the Amenity/Service/Catering option. On clicking Add icon, a window where a list of available amenities, services and catering available for the selected room will be displayed. User can also add additional Amenities, Services or Catering by clicking the Add option displayed at the top right of this window. While adding Amenities/Catering the user can specify if the selected Amenities/Catering is mandatory or optional by selecting the required or Optional check box. The selected Amenity/Service/Catering will be listed in the corresponding options on clicking Save button. The added items can be deleted one by one by selecting the item and clicking the ‘-‘button.

Once all the requirements are entered, the request can be submitted by clicking the Reserve button. The newly submitted request will be confirmed directly without any approval process. Request can be created by any iDrawings Users irrespective of the drawing access set for different User roles.

Reserve a Seat

New Seat Booking Requests can be created directly from the drawing through this option. This option is available under Space Tools icon .

On selecting Reserve a Seat option, all the Hoteling Seats in the floor will be shown blinking in the Drawing and a message ‘Select a HotelingSeat’ will be displayed.New Seat Booking Request page will be displayed on selecting the required hoteling Seat, where the user can select/enter Booked for, Resources and Comments details. Date,Time and Location details will be displayed as read only. The Seats can be booked for iDrawings Users, Employees defined in the Employee module. Booked for field will not be displayed if the logged in User is not an Administrator.

Check Availability button will be displayed to check the availability of the Seat in the selected time slot.

Location button is provided in the New Seat Booking Request window to view the details of the Location which includes the Site, Time Zone, Building, Floor, Space Function, Room No. and Seat No. will be displayed as read only.

Once all the requirements are entered, the request can be submitted by clicking the Reserve button. The request will be confirmed directly without any approval process.

Move a Seat

This option is available under Space Tools icon , to move a Seat within the selected Space. On selecting Move a Seat option, User will be prompted to select a Seat and cursor will turn to cross. Select the Seat to be moved and click another point in the assigned Space (Space where the seat is already assigned). Selected Seat will be moved to specified position in the assigned Space.

Note: Seats cannot be moved to another Space.

Export

Save As

This option is for saving the Drawing into dwg format. Versions of the AutoCAD can be selected while saving the Drawing.

Attachments

Administrators can add/edit/delete/view/download attachments to a Space using this option. Select the required row and click the Attachments option from More icon. The files that can be attached will be restricted to data files like Word, Excel, pdf, jpg, gif, bmp etc.

The File attachments window opens up on selecting Attachments option, showing any files that are already attached to this site. Attachments can now be added to the selected site.

Export

This option is to export the space to the Excel. On clicking Export icon, a confirmation message, “Do you want to export the data along with Space Handle?” with Yes and No buttons will be displayed. On clicking Yes to the message, data will be exported with space handle. Else it will be exported to excel without space handle.

If any Shared Spaces exist, such spaces will be exported as separate rows for each organizational unit in the shared space. All space details except the area details and organizational units will be same for those rows

Keyword Search

This option helps users to search for space details using keywords. The keyword could belong to any data field of the space data. The user need not know or specify which field to look in. The Keyword could be any part of the space data viz. space category, space function, room number, cost category, area, etc. or the additional data fields.

On entering the keyword and clicking on the Search icon, the data grid will be filtered with space records having data matching the specified keyword.

A facility user would be able to view only those floors/divisions for which privileges are granted to him.

Advanced Search

This option helps users to search for space details with specific data in known data fields. The data being searched for can be selected from the list boxes or entered into text boxes provided against the data fields. The user can select one or more data fields viz. space category, space function, room number, cost category, area, etc. or the additional data fields. After entering the search criteria and click Search button, the data grid will be filtered with space records which satisfy the selection criteria.

Fields to be displayed in the Search window can be set using the Configure icon, provided in the Search window.

Search criteria entered in the Search window can be cleared by clicking the Clear button provided in the Search window. After clearing the Search criteria and click Search button to clear the Search Results in the Space Data grid.
