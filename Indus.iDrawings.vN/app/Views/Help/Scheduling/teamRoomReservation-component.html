This option is used to create Team Room Reservations and book space for meetings or other events. Through this option it is possible to check the availability of space and ensure that there are no conflicting bookings. This option will be available only if Enable Team Room Booking option is enabled for the customer. Reservations can be submitted for booking the required meeting rooms.

Reserve Team Room and Reservations are provided under the Team Room Reservation menu. Two tabs namely My Reservations and Active Reservations are provided in the Reservations menu.

Reserve Team Room option helps users to book team rooms for meetings or events. Reservations can be submitted for booking the required rooms.

My Reservations option is for processing Reservations created for the logged in User. User can modify or cancel the Reservations or can send reminder mail to the invitees through My Reservations tab.

Active Reservations tab will be available only for Administrators and Concierges (Division Administrators). For an Administrator, all the Reservations made by different Users will be listed in the Active Reservations tab. All the Reservations made by the concierges or Reservations made in the accessible Site and Team will be listed for a Concierge in the Active Reservations tab.

Reserve Team Room

The user can reserve a Team room through the option Team Room Reservation: Reserve Team Room. On selecting this option, all the Team rooms available for Reservation for the current date will be displayed by default. Team Rooms available for booking are those for which the Space Assignment Type is selected as Scheduling Space.

User can change the date displayed as required. By default, the time slot chart will be displayed with half hour slots for each room starting from 6:00 AM to 7:00 PM. Already The time slots which are already booked, will be shown as highlighted and on pointing the mouse of the highlighted cell; booking details will be displayed at the tool tip. The time slots that cannot be booked will be displayed as inactive and cannot be selected.

A Filter link is provided to filter the workspaces based on Site, Building, Floor, Room Number and Space Function.

The user can view the location of a room in the floor plan by clicking the Floor link in the listed Team Rooms. On clicking this link, the Floor plan of the selected floor will be opened with the available rooms as hatched and its tool tip will show the space details.The selected room will be shown in green color and other rooms which can be scheduled will be shown in yellow color.

The Room Availability Calendar with Room No. link and time slots representing the details of the highlighted room will be displayed below the floor plan.

On clicking the Room No. link, selected Room will be shown as blinking in the Floor Plan and a Room Details window will be displayed with all the details of the selected room. Any photo added for the room will also be displayed in the Room Details window.

The reservation of the required room can be done by clicking the required time slot in the time slot chart displayed in the Reserve Team Room page as well as through the time slot chart in the Floor link. On clicking a time slot, New Team Room Reservation page will be displayed where the User can enter/select From (Date time), To (Date time), Event Name, Participants (Count), Booked for, Invitees and Comments.

On clicking the Location button, the details of the Location which includes the Site, Building, Floor, Space Function and Room No. will be displayed as read only. The requester can specify the number of participants and also select Invitees for the meeting. Invitees can be selected from iDrawings Users.

Add Invitees: Invitees can be added by clicking the Add button placed at the right side of the Invitees list box. On clicking the Add button, Invitees page will be displayed to select the invitees. iDrawings Users and Employees are the options listed in the User Category combo. User has to select the required User Category from the combo and all the Users of selected User Category will be listed. Select the required Invitees and click Save button. Selected Invitees will be added.

Invitees can be removed from the list by clicking the Remove button after selecting the Invitee to be deleted.

Once all the requirements are entered, the request can be submitted by clicking the Reserve button. The newly submitted Reservation will be listed in the My Reservations tab of the Booked for User.

Reservation can be made by any Users. All Users except a Concierge can make reservation for themselves only. A Concierge can make reservation for users in his / her Team or Division.

This options list the Team Room Reservations created for the Customer. Two tabs namely My Reservations and Active Reservations are provided in Reservations menu.

My Reservations

This option is to list all the Team Room Reservations booked for the logged in user. Through this option the user can modify, set reminder or cancel the reservations.

All the active Reservations will be listed with 'Confirmed' status. A checkbox "Show active reservations" is provided in the My Reservations tab to show only Reservations in 'Confirmed' or 'Checked In' status. By default, the checkbox will be displayed as selected.

Reservation will be moved to 'Auto Bump' status if the Reservation time is lapsed and the User has not checked in the Reservation. If the User has checked in the Reservation from Hoteling App or KIOSK, and Reservation time is lapsed, the Reservation will be moved to 'Expired' status.

Modify Reservation: This option is to modify a Team Room Reservation. The logged in User can modify the reservation if the Reservation is created by logged in User and if the Reservation is in Confirmed Status. After modifying the Reservation, Meeting Rescheduled Mail will be sent to all Invitees.

Note: Reservations in Confirmed status only can be modified.

Reminder: This option is used to send reminder to the invitees in the selected scheduled Reservation. Reminder can be set for a Reservation by clicking the Reminder icon. User can send reminder to the invitees only if the selected Reservation is in Confirmed status.

Cancel Reservation: A Reservation can be cancelled by clicking the Cancel Reservation icon. User can cancel the Reservation in Confirmed status only. While cancelling the Reservation, status of the selected Reservation will be changed to Cancelled. If a Confirmed Reservation is cancelled, Meeting Cancelled mail will be sent to all invitees in the Reservation and to the Requester.

Invitee Status

The user can view the status of the invitees using this option. On selecting this icon, an Invitee Status window will be displayed with the Event name as the caption and, other details like when, where and who all are attending the meeting will also be listed in the window. The text of each invitee's names will be displayed in Green, Red or Black color as per the response to the invitation mail send to each of them. If the invitee has accepted the invitation then the text color of the name will be displayed in Green color. If the invitee has declined the invitation then the text color of the name will be displayed in Red color and if the invitee has not taken any action in response to the invitation then the text color of his/her name will be displayed in black color in the Invitee Status window.

Active Reservations

This option is used for viewing all the Team Room Reservations submitted by different users. This tab will be available only for Administrators and Division Administrators (Concierges). For an Administrator, all the Reservations made by different Users will be listed. All the Reservations made by the concierges or Reservations made in the accessible Site and Team will be listed for a Concierge.

A checkbox "Show active reservations" is provided in the Active Reservations tab to show only Reservations in 'Confirmed' or 'Checked In' status. By default, the checkbox will be displayed as selected.

Through this option the user can modify, set reminder or cancel the reservations or see the Invitees for the Reservation.

By default, the Reservations will be listed in the ascending order of 'From' and 'To' dates of the Reservations. The list can be sorted on any column by clicking the column header.

Modify Reservation:This option is to modify a Team Room Reservation. The logged in User can modify the reservation if the Reservation is created by logged in User and if the Reservation is in Confirmed Status. After modifying the Reservation, Meeting Rescheduled Mail will be sent to all Invitees.

Note: Reservations in Confirmed status only can be modified.

Cancel Reservation: This option is to cancel the Reservations which are in confirmed status. Select the Reservation to be cancelled and click Cancel Reservation icon, the selected Reservation will be cancelled. On Cancelling the Reservation, Invitation Cancelled mail will be sent to all selected Invitees in the Reservation and to the Requester.

Check In and Check Out can be done by the Booked for Users and this option will be available in Mobile or KIOSK application.
