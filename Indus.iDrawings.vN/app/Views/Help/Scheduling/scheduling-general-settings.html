This option allows the administrator to select the floor drawings for Scheduling. The floors defined by the administrator and selected for Space management are displayed in the site-building-floor hierarchy with select boxes for each floor. The floors which are selected here will be available in the Scheduling Module for managing the hoteling Seats and scheduling Rooms.

Sites/Buildings/Floors can be selected by selecting the select boxes against each and then clicking Save Changes button.

Amenities comprise furniture like Conference Table, Podium, etc., Audio Visual equipment like bulletin boards, microphones, Overhead Projection system etc. that arerequired in meeting rooms.

This option allows the administrator to add, edit and delete Room Amenities. Amenities added will be available for the user while reserving the Rooms.

Add: Create a new amenity by clicking the Add icon. A new Amenity window will be displayed on clicking Add icon. The user can enter the name of the Amenity and Description and click Save button.

Edit: Edit an Amenity by selecting it and then clicking the Edit icon.

Delete: Delete an Amenity by selecting it and then clicking the Delete icon.

Services list all the services offered in the scheduled room; which can be IT Services, Cleaning Services, Security services etc.that are required in meeting rooms.

This option allows Administrator to add, edit and delete services.

Add: Create a new Service by clicking the Add icon. A new Service window will be displayed on clicking Add icon. The user can enter the name of the Service and Description and click Save button.

Edit: Edit a Service by selecting it and then clicking the Edit icon.

Delete: Delete a Service by selecting it and then clicking the Delete icon.

Catering option lists all the catering services offered in the meeting room and its details which include the Catering Item, Rate and Description. This option allows the administrator to add, edit and delete Catering Items.

Add: Create a new Catering Item by clicking the Add icon. A new Catering Item window will be displayed on clicking Add icon. Enter Catering Item, Rate and Description and click Save button.

Edit: Edit a Catering Item by selecting it and then clicking the Edit icon.

Delete: Delete a Catering Item by selecting it and then clicking the Delete icon.

Different types of resources associated with seats such as Laptop, Phone, LAN Connection etc. can be defined through this option. The resources defined through this option will be available to the user while booking seats.

This Resources option allows the administrator to add, edit or delete Resources.

Add: Create a new Resource by clicking the Add icon. A new Resource window will be displayed on clicking Add icon. The user can enter the name of the Resource and Description and click Save button.

Edit: Edit a Resource by selecting it and then clicking the Edit icon.

Delete: Delete a Resource by selecting it and then clicking the Delete icon.
