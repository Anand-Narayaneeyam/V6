This option is available under Scheduling Module icon. When selecting the Scheduling icon, all the menu options for Scheduling Module will be displayed in the Left panel.

This section provides an option for the users to view the floor plans enabled for Scheduling Floor Plan Management. List of all Floor Plans will be displayed, organized by Site/Building/Floor, with the Floor Plan File Names and File Size. This list can be sorted on any column by clicking on the column header.

View Floor Plan

On selecting a Floor Plan and clicking on the View icon, the Floor Plan will be opened in a new window with a horizontal menu bar and data in the right side. The Space Data is fetched from the database and displayed in the Floor Plan in dynamically created layers. Only the Space Polyline and Space Data layers will be turned on by default.

The Scheduling data namely Workspaces, Team Rooms and Special Use Rooms corresponding to the selected Floor Plan also will be displayed in separate tabs on the right side of the Floor Plan. On bringing the mouse pointer over a space, a Tooltip will be displayed, showing the space data associated with the space

The Right Click menu options provide an easier access to the frequently used functionalities of the Scheduling module for the users, in turn saving time and effort.

On right clicking the mouse inside a Space in the floor plan, a pop-up menu will appear with the options Set as Hoteling Workspace, Set Multiple Rooms as Hoteling Workspace, Edit Space, Assign Space Standard, Select Multiple Spaces by Window, Pan, Zoom in, Zoom Out, Zoom Window, Zoom Extents and Layers

A brief description on the Right click options provided in Floor Plans of the Scheduling module is provided below.

Set as Hoteling Workspace:This option is available only if the Workspace Booking option is enabled for the Customer. This option is for assigning a Space as Hoteling Workspace and the option will be available in the right click only if the selected space is not assigned with any Space Assignment Type.

Right click on a Space and select Set as Hoteling Workspace from the menu options, selected Space will be assigned with Space Assignment Type – Hoteling and one Hoteling Workspace will be added to the Space.

Note: This menu option will be available only if the selected Space is not assigned with any Space Assignment Type.

Set multiple Rooms as Hoteling Workspace: This option is available only if the Workspace Booking option is enabled for the Customer. This option is for assigning the selected Spaces as Hoteling Workspaces.

Right click on a Space and select the Set multiple Rooms as Hoteling Spaces option. Cursor turns to cross and wait for space selection. Select the required Spaces and then right click, all the selected spaces will be assigned as Hoteling Workspaces and one Hoteling workspace will be added to each Space.

Note: This menu option will be available only if the selected Space is not assigned with any Space Assignment Type.

Book Workspace: On Right clicking a Hoteling Workspace, Book Workspace menu will be displayed along with the other default menu options. This option is for creating a Workspace Reservation.

To Reserve a Workspace directly from the Drawing, right click a Hoteling Workspace. Then select Book Workspace menu. Reserve Workspace page will be displayed. Enter the Reservation details and click Reserve button. Workspace will be reserved for the logged in User.

Book Team Room: On Right clicking a Scheduling Space, Book Team Room menu will be displayed along with other default menu options. This option is for creating a Team Room Reservation.

To Reserve a Team Room directly from the Floor Plan, right click a Scheduling Space. Then select Book Team Room menu. Reserve Room page will be displayed. Enter the Reservation details and click Reserve button. Team Room will be reserved for the logged in User.

Edit Space: This option is for editing the Space Data in the Floor Plans.

Assign Space Standard: This option is for assigning space standard and space assignment typeto any desired space. Space standard can be assigned/de-assigned by selecting the space, to which the space standards is to be assigned. Space Assignment Type can be set to Assignable, UnAssignable, Hoteling, Special Use Room and Scheduling Space. Scheduling Space option will be available only if Team Room Booking option is enabled for the customer.

Special Use Room and Hoteling options will be available only if Workspace Booking option is enabled for the Customer.

Select Multiple Spaces by window: This option is for selecting multiple spaces in the Floor Plan by window

Pan: This option enables the user to pan the Floor Plan. On selecting this option, the mouse pointer will change to a hand symbol. Now click the left mouse button, and without releasing it, move the mouse to pan the Floor Plan.

Zoom In: This option enables the user to Zoom in the Floor Plan.

Zoom Out: This option enables the user to zoom out the Floor Plan.

Zoom Window: This option enables the user to zoom the Floor Plan to a Window selection.

Zoom Extents: This option enables the user to view the Floor Plan fully within the Floor Plan display window by resetting any zoomed or panned states.

Layers: This option enables turning On and Off the layers in the base Floor Plan. The dynamically generated layers are not available here. These have to be controlled through the Display Settings. Clicking on the Default button turns on the default space polyline layers.

The following menu options are available to work with the Floor Plans.

Reserve a Workspace

Workspace Reservation can be made directly from the Floor Plan through this option.

On selecting Reserve a Workspace icon, all the Hoteling Workspaces in the floor will be shown blinking in the Floor Plan and a message 'Select a Hoteling Workspace' will be displayed. Reserve Workspace page will be displayed on selecting the required hoteling Workspace, where the user can select/enter Booked for, Comments, Date and Time details. Room No. will be displayed as read only.

The Booked for field will not be displayed if the logged in User is not a Division Administrator (Concierge). For Users other than Concierge, Bookings can be done only for themselves. Concierge can book for other Users in same division.

Check Availability button will be displayed to check the availability of the Seat in the selected time slot.

Once all the requirements are entered, the request can be submitted by clicking the Reserve button. The request will be confirmed.

Reserve a Team Room

Team Room Reservations can be made directly from the Floor Plan through this option.

On selecting Reserve a Team Room icon, all the scheduling spaces will be shown as hatched and a message 'Select the Space for Scheduling' will be displayed. Select one of the hatched spaces for scheduling by clicking on it. A Month/Time view of the Calendar will be displayed with all the days of the selected Month and time as half hour slots. The time slots with booking will be highlighted in red color. Tool tip is provided for the highlighted slots to show the Event Name and time slots.

Team Room Reservation also can be made by clicking on the required time slot from the calendar view with date and time.

Reserve Room page will be displayed with Reserve Room title where the User can enter/select From (Date time), To (Date time), Event Name, Participants (Count), Booked for, Invitees and Comments.

Check Availability button will be displayed to check the availability of the Room in the selected time slot.

The requester can specify the number of participants and also select Invitees for the meeting.

The user can add Invitees as required while reserving the room. Invitees can be selected from iDrawings Users list.

Add Invitees: Invitees can be added by clicking the Add icon placed at the right side of the Invitees list box. On clicking the Add icon, Invitees page will be displayed to select the invitees. User has to select the required User Category from the combo and all the Users of selected User Category will be listed. iDrawings Users will be listed in the User Category combo. Select the required Invitees and click Save button. Selected Invitees will be added. Invitees can be removed from the list by selecting the Invitee to be removed and then clicking the Remove button.

Once all the requirements are entered, the request can be submitted by clicking the Reserve button. The newly submitted request will be listed in the My Reservations tab of the Booked for User. Request can be created by any iDrawings Users irrespective of the Floor Plan access set for different User roles.

Move a Workspace

This option is to move a Workspace within the selected Space. On selecting Move a Workspace icon, User will be prompted to select a Workspace and cursor will turn to cross. Select the Workspace to be moved and click another point in the assigned Space (Space where the seat is already assigned). Selected Workspace will be moved to the specified position in the assigned Space.

Note: Workspaces cannot be moved to another Space.

Set Multiple Rooms as Hoteling Workspace

This option is for assigning the Spaces as Hoteling Workspaces by selecting multiple spaces in the Floor Plan by window.

Click on the Set Multiple Rooms as Hoteling Workspace icon, a toast message 'Select a window, click to select/deselect space, Right Click to exit' is shown. Then the mouse pointer becomes a cross. Select a window by clicking and dragging inside the Floor Plan. The selected spaces will be hatched. Right Click when all the required spaces are selected,, all the selected spaces will be assigned as Hoteling Workspaces and one Hoteling workspace will be added to each Space.

Set as Hoteling Workspace

This option is for assigning a Space as Hoteling Workspace. Click on the Set as Hoteling Workspace icon, a message "Select a Space" is shown. Select a Space and the selected Space will be assigned with Space Assignment Type – Hoteling and one Hoteling Workspace will be added to the Space.

Distribution Map

This option is available only after clicking the Distribution Map icon

This option allows users to get the distribution of space areas across different Organizational Units (Divisions, Departments, Cost Centers etc as defined in the Organizational structure) or based on the Validated Fields. The Spaces belonging to different Organizational Units or the Validated Fields are hatched in the Floor Plan with different colors and hatch pattern. Users can select the Organizational Units or the validated fields for which the distribution is required.The color displayed according to the settings in Settings: Space: General Settings: Distribution Map Settings and can be changed by the user if required.

Space Tools

All the Space related operations such as Edit Space, Assign Space Standard, Move Legend, Select Multiple Spaces by Window, Totalize, Measure Distance and Measure Area are grouped under Space Tools icon .

Edit Space

This option is to edit the Data of the selected Space. On selecting Edit Space option from Space Tools icon, user will be prompted to the select the space to be edited and the cursor turns to cross. Edit Space Data window will be displayed on clicking the required Space in the Floor Plan. Make necessary changes and click the Save button. Space Data edit is possible only to users having Edit privilege.

By default, Space Category is enabled. If the value selected is one of the Common Area space categories, like Building Common, Floor Common or Vertical Space it will not be possible to edit/set the Organizational Unit for this space, and all Organizational level combo boxes will be disabled. But if it is required to assign Organizational Units to Common Spaces also, Organizational Unit Assignment to Common Area can be enabled for the customer.

Data for additional data fields can also be entered through this data entry window.

Assign Space Standard

This option is for assigning space standard and space assignment type to any desired space. On selecting Assign Space Standard option from Space Tools icon, user will be prompted to the select the space for assigning the Space standard and the cursor turns to cross. Assign Space Standard window will be displayed on clicking the required space in the Floor Plan. Space Standard can be assigned/de-assigned to the selected Space. Space Assignment Type can be set to Assignable, Unassignable, Hoteling, Special Use Room and Scheduling Space. Scheduling Space option will be available only if Team Room Booking option is enabled for the customer. Hoteling and Special Use Room options will be available only if Workspace Booking option is enabled for the customer

Assignable:

If Workspace Booking is enabled for the customer and the Space Assignment Type is selected as Assignable, a link named 'Seats' will be displayed against Room Seating Capacity option to define seats for the selected space. On selecting this link, Seats window will be displayed listing all the available seats (if any).

User can add or delete single/multiple seat(s) to/from the selected space using the following options listed in the Seats page. An option to set seats as Hoteling Seats also will be displayed in the Seats list page.

Different options such as with Prefix, With Room Number and without Prefix are provided to set the Seat Number while adding the Seats.

Add Single Seat: The user can add seats to the selected space, one at a time using this option. To add a single seat to the selected space, the user has to select Add icon and select 'Single' radio button. Then options to specify the Prefix and the Number of the specific seat will be displayed in the Add Seat window. The number of seats that can be added is limited to the Minimum Seating Capacity assigned to the selected space.

Add Multiple Seats: The user can add multiple seats at a time to the selected space using this option. To add multiple seats to the selected space, the user has to select Add icon and select 'Multiple' radio button. Then options specify the Prefix of the seats and the number range in which the seats are to be added in the Add Seat window. The number of seats that can be added is limited to the Minimum Seating Capacity assigned to the selected space.

All the Seats set as Hoteling Workspaces can be booked by Users of all User Roles.

Delete Seat: Seats added can be deleted by selecting the required seat and then clicking Delete icon.

Update: Seats can be set as Hoteling Seat by selecting the checkbox of the required Seat and then clicking Update button.

Hoteling:If Space Assignment Type is selected as Hoteling and a Room Seating Capacity is given, Hoteling Workspaces (equal to the count of Room Seating Capacity) will be added to Space. These Workspaces can be booked for Workspace Reservations. These Workspaces can be booked by Users of all User Roles.

Special Use Rooms: If Space Assignment Type is selected as Special Use Rooms and a Room Seating Capacity is given, Special Use Rooms (equal to the count of Room Seating Capacity) will be added to Space. These Workspaces can be booked for Workspace Reservations by the Concierges only.

Scheduling Space: If Space Assignment Type is selected as Scheduling Space, an option to select the Amenities of the room and its count will be displayed. A Team Room can be booked only if the Space Assignment type is assigned as Scheduling Space.

Move Legend

This option is to move a legend from one point to another. Select Move Legend option from Space Tools icon , a toast message will be displayed, prompting 'Select an insertion point to add legend’. Then the mouse pointer turns to cross, and click on the location where the legend is to be moved.

Select Multiple Spaces by Window

This option is for selecting multiple spaces in the Floor Plan by window, for viewing details of space data linked to those spaces.

Click on the Select Multiple Spaces by Window option from Space Tools icon , a toast message 'Select a window, click to select/deselect space, Right Click to exit' is shown. Then the mouse pointer becomes a cross. Select a window by clicking and dragging inside the Floor Plan. The selected spaces will be hatched. Right Click when all required spaces are selected, the Data grid in the Floor Plan page will be highlighted with the data of the selected spaces.

A selected space can be deselected if required, by clicking on the De-Hatch option in the Right Click or by clicking Tools: De-Hatch option.

Totalize

This option is to show the number of spaces selected, Gross Area Total & Net Area Total of the selected spaces.

Select the spaces to be totalized by Select Multiple Spaces by window and click Totalize option from Space Tools icon , a message box will be displayed with details like Number of Spaces selected, Gross Area Total, etc. of the selected spaces.

Apart from this, if any additional data fields of Space are set as Totalizable, the total value of such fields also will be shown.

The details shown in Totalize window can be added as legend by clicking the Add Legend button in the Totalize window.

Tools

All the Floor Plan related operations such as Display Settings, De-hatch and Stop Blink are grouped under Tools icon

Display Settings

This option is for specifying the Space Data fields to be displayed in the Floor Plan and the tooltip. Changes made here will be saved as user default settings and will be effective whenever the User opens any Floor Plan.

For making permanent changes, an administrator has to click Yes to the confirmation message "Display Settings updated. Do you want to save this as System default?"

Display Settings option is available under Tools icon.

De-Hatch

This option is to de-hatch the hatched Space (if any) in the Floor Plan. De-hatch option is available under Tools icon.

Stop Blink

This option is to stop the blinking of the hatched Space (if any) in the Floor Plan. Stop Blink option is available under Tools icon.

Maximize

The width of the Floor Plan and Data tab sheet can be increased / decreased by clicking the Arrow icon provided above the top right corner of the Floor Plan window. User can change the view mode to full screen by clicking the icon provided above the top right corner of the Floor Plan window. User can exit the full screen to normal view by clicking the Maximize icon provided above the top right corner of the Floor Plan

Data

Space data of the all the Workspaces, Team Rooms and Special Use Rooms added to the selected Floor Plan are provided as separate tabs at the right hand side of the Floor Plan. Options - Show Floor Plan, Show Zoomed and Search are provided in the each tab.

Workspaces

Workspaces tab lists all the Rooms with Space Assignment Type - Hoteling. It also lists the Spaces with Space Assignment Type – Assignable and having Hoteling Seats defined through Seats link.

Team Rooms

Team Rooms tab lists all the Rooms with Space Assignment Type - Scheduling Space.

Special Use Rooms

Special Use Room tabs list all the Rooms with Space Assignment Type - Special Use Rooms.

Following options are available in all Data tabs.

Show in Floor Plan

This option is for showing the selected Space in the Floor Plan. Select the Space record from the Data Grid and click on the Show in Floor Plan icon. The Floor Plan will be displayed with the selected Space hatched.

Show Zoomed

This option is for showing the selected Space in Floor Plan in the zoomed in position. Select the Space record and click on the Show Zoomed icon. The Floor Plan will be displayed in the zoomed in position with the selected space hatched.

Keyword Search

This option helps users to search for space details using keywords. The keyword could belong to any data field of the space data. The user need not know or specify which field to look in. The Keyword could be any part of the space data viz. space category, space function, room number, cost category, area, etc. or the additional data fields.

On entering the keyword and clicking on the Search icon, the data grid will be filtered with space records having data matching the specified keyword.

A facility user would be able to view only those floors/divisions for which privileges are granted to him.

Advanced Search

This option helps users to search for space details with specific data in known data fields. The data being searched for can be selected from the list boxes or entered into text boxes provided against the data fields. The user can select one or more data fields viz. Date, Space Function, Room Number and Status. After entering the search criteria and click Search button, the data grid will be filtered with space records which satisfy the selection criteria.

User can check for the Available /Booked Rooms in the selected time.

Search criteria entered in the Search window can be cleared by clicking the Clear button provided in the Search window. After clearing the Search criteria and click Search button to clear the Search Results in the Space Data grid.

If any prefix has been set for Room Number for the customer, then a read-only prefix will be displayed before the Room Number Text box. Only the Room Number need be entered, the prefix will be automatically inserted.

Share Space

The Administrator can view/add/edit/delete the sharing of space using this option. The Sharing among different Organizational Units should sum up 100.

If the selected Space is already shared, it will be indicated with a caption ‘Shared’ next to the Organizational Unit name. Click 'View' link to view the existing sharing details. Click View link, the Space Sharing by Organizational Units window opens up with the percentages of already assigned Organizational Units.

Add: On clicking Add icon, Assign Shared Organizational Unit window will be displayed with Org. Units combo boxes and Percentage textbox. Select the required Organization Unit name from the required level and enter the percentage. The sharing details will be saved by clicking Add button.

Edit: Space sharing can be edited by clicking the Edit icon. The Sharing among different Organizational Units should sum up to 100.

Delete: Remove an assigned Organizational Unit by selecting the same and clicking the Delete icon.
