This option allows the Administrator to view a log of user activities in the Scheduling Module. The period for which the log is to be viewed can be specified, and the users, entities and activities can be selected. This option will be available only if the ‘Logbook’ option is enabled for the customer.

To view the log activities in the Scheduling Module, user has to select the Scheduling Module from the Module list in the Log book Filter option. Log activities can also be filtered based on Users, Entities and Activities.
