Data Field Users can create Additional Data Fields to the Employee and Move Project data in addition to the system defined data fields which are available by default. For creating a new Data Field, the Add and Edit options are provided. The Data Field Name, Data Field Type (Text Field, Text Area, Date, Date Time, Integer or Numeric), whether Mandatory or not, whether the field is Totalizable or not and whether a Validated field or not etc. have to be specified. If the selected data type is of Text Area, the validated property will not be available. If the selected data field is of type Numeric, then only the Totalizable option is available. If a data field is set as Totalizable, the total value of that field will be displayed in reports (Subtotal and Total) as well as in Totalize window. For Validated fields, the set of allowed values will also have to be specified through the Field Values option. For e.g. Marital Status could be a validated field for Employee with values Single, Married, Divorced, and Separated. For Mandatory fields, it would be required to enter a value for it while entering Employee or Move Project data.

Custom Reports are reports created by the users according to their own requirements. Users can create and save reports with a suitable name through the Custom Reports option under Settings: Space: General Settings. It is also possible to share these reports with other users of the system.

Users can view and print these reports whenever required by selecting them from the Custom Reports option under Reports. Reports created by other users and shared will be listed with the name of the owner displayed in parenthesis after the report name

The Custom Reports option under General Settings lists all the custom reports created by the user.

Add: Create a new Custom Report by clicking the Add icon.

On clicking the Add icon, a report column selection window pops up with buttons for Field Order, Preview and Save. All the Data fields that are available will be listed in alphabetic order with check boxes.

The required columns for the Report can be selected by checking the corresponding data fields. Fields once selected can be deselected by unchecking them. Group Fields can be specified for grouping the data rows of the report. Each group will be printed with a Sub heading showing the group field values of the group. Show Subtotal, Show Grand Total also can be set for Numeric/Integer type static fields like Area, Cost, etc. and Numeric/Integer type additional data fields which are set as totalizable in the setup.

Field properties like Column Heading and Column Width (in Inches) can also be set. By default, the Report Column Headings will be the same as the Data Field Names. Users can change a Report Column Heading by changing the Field Name in the Column Heading. The width of a Report Column is set as 1.5 inch by default. User can change the Report Column Width of the required Data Fields.

Field Order: The order of the Report Columns and Sub Header Fields can be set through this option. On clicking Field Order button, Field Order window pops up with all the Header and Sub Header fields. Each data field can be moved by selecting and dragging it to the required position.

Preview: Users can view the report using this Preview option. After selecting the report fields and setting the Field Order the report can be previewed by clicking the Preview button. A Report Heading can also be given for viewing the report. The Paper orientation can be set to Landscape or Portrait by clicking on the corresponding radio button.

The report can be shared with other users by checking the corresponding check box.

User can go to the previous stages and edit the settings already specified using the corresponding buttons.

Save: User can save the report by clicking the Save button. Once a new report is created and saved by a user, it will be listed in the Custom reports in Reports menu.

Edit: Edit a Custom Report by selecting it and then clicking the Edit icon. The Report Columns, field order, report name, etc. can be modified through this option.

Report column selection window pops up with buttons for Field Order and Preview, Save As and Save, on clicking Edit icon. All the Data fields that are available will be listed in alphabetic order with check boxes and all the data fields selected for the report will be displayed as checked.

Preview: After making the modifications, Users can view the report using this Preview option.

Save: User can save the changes in the report by clicking the Save button.

Save As: User can make changes to an existing report and save it as a new report with a different name using this option, by clicking the Save As button.

Delete: Delete a Custom Report by selecting it and then clicking the Delete icon.
