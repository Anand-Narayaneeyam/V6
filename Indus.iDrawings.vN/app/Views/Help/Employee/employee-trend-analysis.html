Through this option Administrator can create Trends on different Entities of Seating capacity and Occupancy

The Trend Analysis shows the trends of Seating Capacities based on Organizational Unit, Seating Capacity based on Space Standard, Grade wise Occupancy by Time and Rate of Hiring by Time. This is available only to the Administrators if the option ‘Employee Trend Analysis’ is enabled by Indus.

The trends are based on data from the Snapshots saved by the user. Snapshots can be taken periodically or whenever there is a significant change in Space/Employee assignments. If Snapshot is created in different months during a year, the Trend details shown will be based on the Snapshot of the last month.

Snapshots

To see the Trends or track changes in the Organizational Unit wise Seating Capacity, Space Standard wise Seating Capacity and Grade wise Employee Occupancy in an Organization, users need to save Snapshots of data as and when changes happen. It should be a deliberate action taken by the user when he/she feels that a particular space/employee change is significant.

A Snapshot saved on a particular date, will have the Seating Capacity, Occupied Seats, Free Seats and Excess Seats details of all the Organizational Units. Snapshots will be created for spaces only if Organizational Unit is assigned to that space and at least one employee placed in it. Snapshots can be exported to selected file formats from the list.

On clicking Trend Analysis: Snapshots,a list of all available snapshots will be displayed. The details of a snapshot can be viewed by selecting the snapshot, and clicking View icon.

Add: Snapshots can be created using Add icon and exported to Excel/pdf/txt/rtf files or saved in the database specifying a Snapshot date.

Delete:Snap shots can be deleted by selecting one snapshot from the list and then clicking the Delete icon.

View: Snap shots can be viewed by selecting one snapshot from the list and then clicking the View icon.

Seating Capacities by Organizational Unit

This Trend will display the Total Seating Capacity of an Organizational Unit based on the Snapshots created. On clicking Seating Capacities by Organizational Unit menu, a report page showing the trend details of all Organizational units based on the Snapshots created.

Seating Capacities by Space Standard

This Trend will display the Total Seating Capacity of a Space Standard based on the Snapshots created. On clicking Seating Capacities by Space Standard menu, a report page showing the trend details of all Space Standards and its Seating Capacities will be displayed with respect to the Snapshots created.

Occupancy by Grade

This trend will display the total number of employees in each grade during a year based on the snapshot created for a time period. On clicking Occupancy by Grade menu, a report page showing the trend details of Employee Grades and Number of Employees will be displayed based on the Snapshots created.

Hiring Rate

This Trend will display the Total Employee hiring based on the Employee Organizational Unit. On clicking Hiring rate Rate menu, a report page showing Number of Employees in each Organizational Units based on the Hiring Year will be displayed.
