This option is available under Employees Module icon. When selecting the Employees icon, all the menu options for Employee Module will be displayed in the Left panel.

This section provides an option for the users to view the drawings enabled for Employees Management, along with the Employee information.

List of all drawings will be displayed, organized by Site, Building and Floor, with the Drawing File Names and File Size. This list can be sorted by clicking on any column by clicking on the column header.

Open Drawing

On selecting a drawing and clicking on the View icon, the drawing will be opened in a new window which is like a Tab sheet and by default it opens the drawing of the currently selected floor of the currently selected Module. All drawing related operations are provided with a horizontal menu bar.

The Space data and Employee data are fetched from the database and displayed in the drawing in dynamically created layers. Only the Space Polyline and Space Data layers and any predefined default display layers will be turned on by default.

The Employee data corresponding to the selected Floor drawing will be displayed on the right side of the Drawing in a tab sheet labeled as ‘Assigned’. All unassigned Employees are displayed in tab sheet labeled as ‘Unassigned’.

If any Employees are scheduled for moving to this floor through Space Planning Move Project, those Employee details will be displayed in ‘Scheduled Moves’ tab.

If the user has no access permission to a Division, the Employees placed in spaces belonging to that Division will not be displayed and data of those Employees also will not be listed

The width of the Drawing can be increased by clicking the Arrow icon provided at the left side of the Drawing window.

On bringing the mouse pointer over an employee symbol, a Tooltip will be displayed showing the data associated with the employee. If the mouse pointer is brought over a blank space, a Tooltip will be displayed, showing the space data associated with the space.

If the user has no access to a Division, the employees in spaces assigned to that Division will not be displayed and data of those employees also will not be listed.

On right clicking the mouse inside the drawing display window, a popup menu will appear with the following options

Edit Space: This option is for editing the Space Data in the Drawings.

Assign Space Standard: This option is for assigning space standard and space assignment type to any desired space. Space standard can be assigned/de-assigned by selecting the space, to which the space standards is to be assigned. Space Assignment Type can be set to Assignable, UnAssignable, and Scheduling Space. Scheduling Space option will be available only if Scheduling option is enabled by Indus.

Select Multiple Spaces by window: This option is for selecting multiple spaces in the drawing by window.

Pan: This option enables the user to pan the drawing. While selecting this option, the mouse pointer changes to a hand symbol. Now click the left mouse button, and without releasing it, move the mouse to pan the drawing.

Zoom In: This option enables the user to zoom in the Drawing.

Zoom Out: This option enables the user to zoom out the Drawing.

Zoom Window: This option enables the user to zoom the Drawing to a Window selection.

Zoom Extents: This option enables the user to view the Drawing fully within the drawing display window by resetting any zoomed or panned states.

Layers: This option enables turning On and Off the layers in the base drawing. The dynamically generated layers are not available here. These have to be controlled through the Display Settings. Clicking on the Default button turns on the default space Polyline layers.

The following menu options are available to work with the drawings.

Distribution Map

This option allows users to get the distribution of space areas across different organizational Units (Division/Department, Cost center, etc. as defined in the Organizational structure), or based on the Validated Fields. The Spaces belonging to different Organizational Units or the Validated Fields are hatched in the drawing with different colors. Users can select the Organizational Units or the validated fields for which the distribution is required.

The colors are displayed according to the default settings set in General Settings: Distribution Map Settings.User can be change the color by clicking the Settings icon provided against each Validated Field. Settings made here will be applicable for the current instance. A legend based on the distribution map also gets inserted automatically with Color of each Organizational Unit and its Area Sq. ft.

Occupancy

This option shows the distribution map based on the Occupancy (Vacant Rooms and Occupied Rooms). While showing this distribution map, the color settings will be based on the user’s Color Preferences. On clicking Occupancy,all the occupied and vacant rooms will be hatched and the corresponding legend showing the area of Occupied and Vacant rooms in Sq. Ft. with color and hatch patterns will be displayed.

Space Tools

All the Space related operations like Move Legend, Select Multiple Spaces by Window, Totalize, Measure Distance and Measure Area in Employee Drawing are grouped under Tools icon .

Edit Space

This option is to edit the Data of the selected Space. On selecting Edit Space option from Space Tools icon, user will be prompted to the select the space to be edited and the cursor turns to cross. Edit Space Data window will be displayed on clicking the required Space in the drawing. Make necessary changes and click the Save button. Space Data edit is possible only to users having Edit privilege.

By default, Space Category is enabled. If the value selected is one of the Common Area space categories, like Building Common, Floor Common or Vertical Space it will not be possible to edit/set the Organizational Unit for this space, and all Organizational level combo boxes will be disabled. But if it is required to assign Organizational Units to Common Spaces also, Organizational Unit Assignment to Common Area can be enabled for the customer.

Data for additional data fields can also be entered through this data entry window.

Share Space

Provision for Sharing a Space between Multiple Organization Units is allowed. This is needed for sharing spaces like Consulting Rooms, Conference Halls, etc. between different departments.

If ‘Space Allocation to Multiple Departments’ option is enabled for the customer, a link ‘Share Space’ will be displayed along with Organizational Unit in the Data Entry window. The Administrator can view/add/edit/delete the sharing of space using this option. The percentage sharing among different Organizational Units should sum up to 100%.

If the selected Space is already shared, it will be indicated with a caption ‘Shared’ next to the Organizational Unit name. Click ‘View’ link to view the existing sharing details. Click View link, the Space Sharing by Organizational Units window opens up with the percentages of already assigned Organizational Units.

Add: On clicking Add icon, Assign Shared Organizational Unit window will be displayed with Org. Units combo boxes and Percentage textbox. Select the required Organization Unit name from the required level and enter the percentage. The sharing details will be saved by clicking Add button.

Edit: Space sharing can be edited by clicking the Edit icon. The Sharing among different Organizational Units should sum up to 100.

Delete: Remove an assigned Organizational Unit by selecting the same and clicking the Delete icon.

Assign Space Standard

This option is for assigning space standard and space assignment type to any desired space. On selecting Assign Space Standard option from Space Tools icon, user will be prompted to the select the space for assigning the Space standard and the cursor turns to cross. Assign Space Standard window will be displayed on clicking the required space in the Drawing. Space Standard can be assigned/de-assigned to the selected Space. Space Assignment Type can be set to Assignable, Unassignable, and Scheduling Space. Scheduling Space option will be available only if Scheduling option is enabled for the customer.

By default “Approval Process: Not required” is selected in Room Booking or Seat Booking. So the Room / Seat booking request will be confirmed directly without any approval process.

If Space Assignment Type is selected as Scheduling Space, an option to select the Amenities of the room and its count will be displayed.A room can be booked only if the Space Assignment type is assigned as Scheduling Space.

If Seat Booking is enabled for the customer and the Space Assignment Type is selected as Assignable, a link named ‘Seats’ will be displayed against Room Seating Capacity option to define seats for the selected space. On selecting this link, Seats window will be displayed listing all the available seats (if any).

User can add or delete single/multiple seat(s) to/from the selected space using the following options listed in the Seats page. An option to set seats as Hoteling Seats also will be displayed in the Seats list page.

Add Single Seat: The user can add seats to the selected space, one at a time using this option. To add a single seat to the selected space, the user has to select Add icon and select ‘Single’ radio button. Then options to specify the Prefix and the Number of the specific seat will be displayed in the Add Seat window. The number of seats that can be added is limited to the Minimum Seating Capacity assigned to the selected space.

Add Multiple Seats: The user can add multiple seats at a time to the selected space using this option. To add multiple seats to the selected space, the user has to select Add icon and select ‘Multiple’ radio button. Then options specify the Prefix of the seats and the number range in which the seats are to be added in the Add Seat window. The number of seats that can be added is limited to the Minimum Seating Capacity assigned to the selected space.

Move Legend

This option is to move the legend added after viewing the Distribution map or Totalize to a user specified location. To move the Legend, Select Move Legend option from Space Tools icon and then click to specify a new location.

Select Multiple Spaces by Window

This option is for selecting multiple spaces in the drawing by window, for viewing details of space data linked to those spaces.

Click on the multiple spaces by window option, from Space Tools icon a toast message ‘Select a window, click to select/deselect Space. Right click to exit’ is shown. Then, the mouse pointer becomes a cross. Select a window by clicking and dragging inside the drawing. The selected spaces will be hatched. Right Click when all required spaces are selected. If any employee/space data linked to this space, the right hand side data grid will be displayed highlighting the data of the employees in the selected spaces. See Data for more details.

A selected space can be deselected if required, by clicking on the De-Hatch option in the Right Click.

Totalize

This option is to display the sum total of Area, number of seats, employees, and any other totalizable data fields in the selected spaces.

Select the spaces for which the employee count and area is to be viewed, and click on the Totalize option from Space Tools option from icon, A window will be displayed with the Number of Spaces selected, Area of the Selected spaces, Maximum Occupancy, Number of Employees, Current Occupancy, Employees in spaces with no seating capacity defined, etc.

No. of Spaces selected – Total number of selected spaces.

Gross Area Total – Total area of the selected spaces.

Maximum Occupancy – The total seats in the selected spaces in which the Space Assignment Type is Assignable .

Current Occupancy – The number of employees in spaces in which the Space Assignment Type is Assignable

Open – This is the number of free seats. (i.e. Maximum Occupancy – Current Occupancy)

Apart from this, if any additional data fields of Space are set as Totalizable, the total value of such fields also will be shown.

The details shown in Totalize window can be added as legend by clicking Add Legend button.

Measure Distance – Point to Point

This option is used to measure the distance between two points. Select Measure Distance - Point to Point option from Space Tools icon and the cursor will turn into a cross. Click on the first point from which the distance is to be measured and then click on the end point, the distance between the two points will be displayed.

Select the option first, and the system will prompt to select the points to Measure Distance. Click the starting point from which the distance is to be measured, and then click the end point. The distance between the two points will be displayed. This marked distance can be removed by clicking the De-hatch option.

Measure Distance – Wall to Wall

This option is used to measure distance between two Walls. Select Measure Distance - Wall to Wall option from Space Tools icon and the cursor will turn into a cross. Click close to the wall from which the distance is to be measured. The starting point will snap to the nearest space polyline (Net Polyline for Net only Customers). Then click close to the other wall to which the measurement is to be taken. The end point will snap to the nearest point of the corresponding space polyline/net polyline. The distance between the two walls will be displayed.

Measure Area

This option is used to measure the area of a polygon in a Drawing. To measure an area, Select Measure Area option from menu and the cursor will turn to a cross. Mark the area to be measured by clicking on any one vertex point, and then clicking on the remaining vertex points one after the other. After the last vertex, close the polygon by right clicking in the drawing. The polygon will be hatched and the area will be displayed. This marked distance can be removed by clicking the De-hatch option.

Employee Tools

All the Employee related operations like De-Assign Employee, Move Single Employee, and Move Multiple Employees are provided in Employee Tools icon

De-Assign Employee

This option de-links the selected Employee from the assigned Space. On clicking this option from Employee Tools menu, a toast message will be displayed, to select the employee to be de-assigned. Then the mouse pointer will turn to cross and select the employee by clicking on the employee. Click OK to the confirmation message then the selected employee will be de-assigned from the drawing.

Move Single Employee

This option is to move employee from one place to other within the same floor. On clicking this option from Employee Tools menu, a toast message will be displayed, to select the employee to be moved. Then the mouse pointer will turn to cross and select the employee by clicking on the employee. Then the selected employee will be hatched and a rubber band like line depicting the move will be displayed. Click on the exact location to which the employee is to be moved.

The Resources can also be moved with the Employee if necessary. For more details,See Move Resources for Single Employee

Move Multiple Employees

This option is to move multiple Employees from one or more space(s) to other space(s) within the same floor. Moving of Employees to places outside the Polyline is not possible. Employees placed in the same or different spaces can be moved to other space(s). Only the Administrator can perform this move. Select Move Multiple Employees option from Employee Tools menu, a toast message ‘Click to select/deselect Employee. Right click to Exit’ will be displayed. Then, the mouse pointer becomes a cross. Select the Employees to moveby clicking on the employee symbol in the drawing. On right clicking the mouse after selecting the required employees, a toast message ‘Select a point inside the space where the employee name is to be located’ appears. Select the exact location where the employee is to be placed (Location can be specified for each employee by clicking the desired location one by one).Selected Employees will be moved to the specified Location.

Move Resources for Single Employee

After selecting the desired location for the employee, a message ‘Do you want to move the employee resources?’ will be prompted with Yes or No buttons to confirm whether the resources attached to the employees are to be moved or not. On clicking ‘No’ to the message, the move resource operation will get cancelled.

On clicking ‘Yes’ to the message, a Resource for Move window will be displayed with the Resources attached to each employee. Select the required resources and on clicking Update the selected resources will be attached with the employee to be moved. After the Single Move operation, the Employee and the selected resources will be moved to the new location.

If the option ‘Allow Move Resources Based on Employee Location’ is enabled by Indus, a Select Resources page will be displayed with all the Resources (Asset/ Furniture) based on the Employee location. If any of the employee resources attached to Employee through Data: Resources menu is not selected while moving, such resources will get delinked from the Employee.

Tools

All the Drawing related operations such as Display Settings, De-hatch and Stop Blink are grouped under Tools icon

Display Settings

This option is for specifying the Space and Employee Data fields to be displayed in the drawing and the tooltip. Changes made here will be saved as user default settings and will be effective whenever the User opens any drawing.

For making permanent changes, an administrator has to click Yes to the confirmation message “Display Settings updated. Do you want to save this as System default?”

De-Hatch

This option is to de-hatch the hatched Space (if any) in the Drawing. De-hatch option is available under Tools icon.

Stop Blink

This option is to stop the blinking of the hatched Space (if any) in the Drawing. Stop Blink option is available under Tools icon.

Maximize

User can change the view mode to full screen by clicking the icon provided above the top right corner of the drawing window. User can exit the full screen to normal view by clicking the icon provided in the drawing toolbar.

Export

This option is for exporting the current view of the drawing to the user’s computer in any of the following formats namely bmp, dwf, svg, pdf etc. The drawing may be exported with the Space data layers, dynamically generated distribution maps, and legends if any.

Save As

This option is for saving the Drawing into dwg format. Versions of the AutoCAD can be selected while saving the Drawing.

Assigned

This option (Data: Assigned Employees) allows the user to get the details of assigned employees in the data grid. Select the required floors, the details of assigned employees in the selected floor can be viewed in the data grid.

Edit

This option is for editing the details of an employee. Select the employee record to be edited and click Edit icon. The details will be displayed. Make the necessary changes and click on Save Changes button.

While editing the employee details the status can be changed from Active to Terminated/Resigned/Retired/Transferred. The Date of Relieving field will be displayed while changing the status to Terminated/Resigned/Retired/Transferred and corresponding date has to be entered. Such employees will get unassigned automatically.

Delete

This option is for deleting the details of an employee. Select the record to be deleted and click Delete icon.

Show Drawing

This option is for showing the selected employee in drawing. Select the employee record and click Show Drawing option from the More icon.The drawing will be opened blinking the selected employee and space hatched.

Show Zoomed

This option is for showing the selected employee in drawing, in the zoomed in position. Select the employee record and click on the Show Zoomed option from the More icon. The drawing will be displayed in the zoomed in position with the selected employee and space hatched.

Resources

This option is for associating Resources (Asset/ Furniture) with an employee. Select the employee record to which the resource is to be added/viewed or deleted, and click on Resources option from the More icon. The resources assigned if any will be displayed with Employee Name & Employee Code as read only. Resources cannot be assigned for unassigned employees if ‘Move Resources Based on Employee Location’ option is enabled by Indus.

Add : New resource can be added by clicking the Add icon.

Make required selections and click Save to associate the resource with the employee. Any number of resources can be associated with the employee.

Delete : One or more assigned Resources can be deleted by selecting it and clicking on the Delete icon.

Display Settings

This option allows the user to change the visibility of the employee and space data fields displayed in the data grid. On selecting this option, all the available space and employee data fields are displayed in different tabs. Display settings can be modified by selecting the required fields from the list. Changes made here will be saved as user default settings and will be effective whenever the User takes the Employee Data grid.

For making permanent changes, an administrator has to click Yes to the confirmation message “Display Settings updated. Do you want to save this as System default?”

Attachments

Users can add/edit/delete/view/download attachments to an employee using this option.Select the required employee and click the Attachments option from the More icon.If any attachments that are already attached will be displayed in a list with Add, Edit, Delete, View and Download icons.

Add: An attachment can be added by selecting the Attachment Category first, and then selecting the file. The attachment has to be uploaded by clicking on the Upload button.

Edit: Edit an Attachment by selecting it and clicking the Edit icon.

Select an attachment and click Edit icon. The Edit Attachment window will be displayed with the current file name. The user can select the Attachment Category and the corresponding file. Changes can be saved by clicking the Save Changes button.

Delete: Delete an attachment by selecting it and clicking the Delete icon.

View: The attachment can be viewed by selecting the file and then clicking the View icon.The selected attachment will be opened in an attachment viewer.

View is supported only for image files.

Download: Attachment can be downloaded and saved in the user's computer by clicking the Download icon.

Export

This option is to export the employee records to the Excel. On clicking Export option from More icon, data will be exported to an excel file.

Keyword Search

This option is available only after clicking the icon.

This option helps users to search for employee details using keywords. The keyword could belong to any data field of the employee. The user need not know or specify which field to look in.

On entering the keyword and clicking on the Search icon, the data grid will be filtered with employee records having data matching the specified keyword. The employees displaying in the grid as a result of search operation will blink in the drawing.

A facility user would be able to view only those floors/divisions for which privileges are granted to him.

Advanced Search

This option helps users to search for employee details with specific data in known data fields. The data being searched for can be selected from the list boxes or entered into text boxes provided against the data fields. Click on the Search button and the data grid will befilteredwith employee records having data matching the specified criteria.

The Search criteria once created will be saved and will be available throughout the current session. For searching with First Name or Last Name, on entering the first character, all employee na.mes starting with that will be listed in the list box provided. User can select the required employee from the list

Fields to be displayed in the Search window can be set using the Configure icon, provided in the Search window.

Search criteria entered in the Search window can be cleared by clicking the Clear button provided in the Search window. After clearing the Search criteria and click Search button to clear the Search Results in the Data grid.

Unassigned

This option (Data: Unassigned Employees) allows the user to get details of unassigned employees. Unassigned Employees are the employees whose locations are yet to be assigned in the drawing.

Add

This option is for entering the details of a new employee into the system. On clicking Add icon, the Employee Data Entry window is displayed, with the predefined data fields and the additional data fields defined by the user. The details of the employee can be entered. Employee photo can be uploaded by browsing and selecting the same. Email of employee is not mandatory, but if entered, it should be unique. No two employees can have the same email id.Also email of Employee should with Mail Domain added in Settings: Administration: General Settings: Mail Domains. While adding an Employee, the Status of the Employee will be set as ‘Active’ by default. On clicking the Save button, the data and the photograph will be saved in the system and a message ‘Employee added’ will be displayed.

Edit
