This option is used to display the details of both assigned and unassigned Employees. Assigned employees will be displayed in data grid with Space/location details like space key, space category, organizational unit fields etc. Unassigned employees will be displayed without Space details and will be displayed with employee fields.

Add

This option is for entering the details of a new employee into the system. On clicking Add icon, the Employee Data Entry window is displayed, with the predefined data fields and the additional data fields defined by the user. The details of the employee can be entered. Employee photo can be uploaded by browsing and selecting the same. Email of employee is not mandatory, but if entered, it should be unique. No two employees can have the same email id.Also email of Employee should with Mail Domain added in Settings: Administration: General Settings: Mail Domains. While adding an Employee, the Status of the Employee will be set as ‘Active’ by default. On clicking the Save button, the data and the photograph will be saved in the system and a message ‘Employee added’ will be displayed.

Edit

This option is for editing the details of an employee. Select the employee record to be edited and click Edit icon. The details will be displayed. Make the necessary changes and click on Save Changes button.

While editing the employee details the status can be changed from Active to Terminated/Resigned/Retired/Transferred. The Date of Relieving field will be displayed while changing the status to Terminated/Resigned/Retired/Transferred and corresponding date has to be entered. Such employees will get unassigned automatically.

Delete

This option is for deleting the details of an employee. Select the record to be deleted and click Delete icon. A confirmation message will be displayed. Click Yes to delete the employee record from the system.

Assign Space

This option is to assign space for an employee in the drawing. On selecting an Employee and then selecting the Assign Space icon, the drawing list is displayed. Select the Drawing to which the Employee has to be assigned, and click the View icon. A message appears prompting the user to select the space where the employee is to be assigned .Click on the exact location where the employee is to be placed.

Resources

This option is for associating Resources (Asset/ Furniture) with an employee. Select the employee record to which the resource is to be added/viewed or deleted, and click on Resources icon. The resources assigned if any will be displayed as Card with Employee Name & Employee Code as read only. Resources cannot be assigned for unassigned employees if ‘Move Resources Based on Employee Location’ option is enabled by Indus.

Add

New resource can be added by clicking the Add icon.

Make required selections and click Save to associate the resource with the employee. Any number of resources can be associated with the employee.

Delete

One or more assigned Resources can be deleted by selecting it and clicking on the Delete icon.

Attachments

Users can add/edit/delete/view/download attachments to an employee using this option. Select the required employee and click the Attachments icon. If any attachments that are already attached will be displayed in a list with Add, Edit, Delete, View and Download icons.

Add: An attachment can be added by selecting the Attachment Category first, and then selecting the file. The attachment has to be uploaded by clicking on the Upload button.

Edit: Edit an Attachment by selecting it and clicking the Edit icon.

Select an attachment and click Edit icon. The Edit Attachment window will be displayed with the current file name. The user can select the Attachment Category and the corresponding file. Changes can be saved by clicking the Save Changes button.

Delete: Delete an attachment by selecting it and clicking the Delete

View: The attachment can be viewed by selecting the file and then clicking the View icon. The selected attachment will be opened in an attachment viewer.

View is supported only for image files.

Download: Attachment can be downloaded and saved in the user's computer by clicking the Download icon.

Display Settings

This option allows the user to change the visibility of the employee and space data fields displayed in the data grid. On selecting this option, all the available space and employee data fields are displayed in different tabs. Display settings can be modified by selecting the required fields from the list. Changes made here will be saved as user default settings and will be effective whenever the User takes the Employee Data grid.

For making permanent changes, an administrator has to click Yes to the confirmation message “Display Settings updated. Do you want to save this as System default?”

Show Drawing

This option is for showing the selected employee in drawing. Select the employee record and click on the Show Drawing icon. The drawing will be opened blinking the selected employee and space hatched.

Show Zoomed

This option is for showing the selected employee in drawing, in the zoomed in position. Select the employee record and click on the Show Zoomed icon. The drawing will be displayed in the zoomed in position with the selected employee and space hatched.

Export

This option is to export the employee records to the Excel. On clicking Export icon, data will be exported to an excel file.

Keyword Search

This option is available only after clicking the icon.

This option helps users to search for employee details using keywords. The keyword could belong to any data field of the employee. The user need not know or specify which field to look in.

On entering the keyword and clicking on the Search icon, the data grid will be filtered with employee records having data matching the specified keyword. The employees displaying in the grid as a result of search operation will blink in the drawing.

A facility user would be able to view only those floors/divisions for which privileges are granted to him.

Advanced Search

This option helps users to search for employee details with specific data in known data fields. The data being searched for can be selected from the list boxes or entered into text boxes provided against the data fields. Click on the Search button and the data grid will befilteredwith employee records having data matching the specified criteria.

The Search criteria once created will be saved and will be available throughout the current session. For searching with First Name or Last Name, on entering the first character, all employee na.mes starting with that will be listed in the list box provided. User can select the required employee from the list .

Fields to be displayed in the Search window can be set using the Configure icon, provided in the Search window.

Search criteria entered in the Search window can be cleared by clicking the Clear button provided in the Search window. After clearing the Search criteria and click Search button to clear the Search Results in the Data grid.
