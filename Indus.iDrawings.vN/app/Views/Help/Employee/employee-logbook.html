This option allows the Administrator to view a log of user activities in the Employee Module. The period for which the log is to be viewed can be specified, and the users, entities and activities can be selected. This option will be available only if the ‘Logbook’ option is enabled by Indus.

Export option is provided to export the log details to an Excel file. Click the Export button.The Save location and the File Format have to be selected, and the File Name has to be entered. The details displayed in the logbook grid will be saved to the user’s computer.
