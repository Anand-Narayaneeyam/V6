The reports generated by the system can be viewed and modified using the controls available in the report viewer. The report can be saved to Excel, PDF or RTF format by clicking on Excel, PDF or RTF button.

Employee Reports are available under Employees: Reports icon.

The following Reports are provided in the Employee Module.

All Employees

This report shows the list of all the Employees.

Assigned Employees

This report shows the details of assigned Employees. On selecting this option, the Select Floors page will be displayed in site/building/floor hierarchy. The required criteria like By Floor, By Division, etc. can be selected from combo, and the required sites/buildings/floors can be selected from the floor list. After making the necessary selection, on clicking Next, Assigned Employees report will get generated based on the criteria selected.

Unassigned Employees

This report shows the details of Employees who are not assigned any space.

Square Footage Analysis

This report shows the details of average square footage analysis. On selecting this option, the Select Floors page will be displayed in site/building/floor hierarchy. From this list, select the required criteria and click Next. The report displays total area in square feet, total number of employees and average square footage.

Detailed Occupancy

For selected Organizational Units/Buildings/Floors, this Report would show the spaces with the area, seating capacity, occupied seats, free seats (if under occupied) and excess seats (if over occupied).

By Organizational Units

On selecting this option, the list of Organizational Units is displayed. Select the required Organizational Unit and click Show Report icon. The report will be generated.

By Site

On selecting this option, the list of Sites is displayed. Select the required Sites and click OK button. The report will be generated.

By Building

On selecting this option, the list of Buildings is displayed, organized by site. Select the required Buildings and click Show Report icon. The report will be generated.

By Floors

On selecting this option, the list of Floors is displayed, organized by site and building. Select the required Floors and click Show Report icon. The report will be generated.

Moves in the Queue

This report shows the list of the moves to be performed. I.e. the moves of employees already assigned/planned, but yet to be moved.

Closed-out Moves

This report shows the list of all closed moves.

Cancelled Moves

This report shows the list of all cancelled (deleted) moves.

Occupancy Distribution - Space Standard

This report shows the occupancy distribution based on Space Standards.

Occupancy Distribution by Space Standard is a consolidated report showing details of Space Standard assigned and its Seating Capacity, Occupied Seats, Free Seats and Excess Seats for the selected Site/Building/Floor.

Occupancy Distribution by Organizational Unit is a consolidated report shows the details of Organizational Unit, Seating Capacity, Occupied Seats, Free Seats and Excess Seats for the selected Site/Building/Floor.

By Site

On selecting the option, the list of Sites is displayed. Select the required Sites and click Show Report icon. The report will be generated.
