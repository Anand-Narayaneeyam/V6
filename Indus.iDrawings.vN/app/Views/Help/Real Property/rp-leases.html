This option is for adding, editing or deleting Leases, Area & Cost Details, Rent Information etc. Options for setting Agreement Clauses, Cancellation Clauses and Renewal of Leases are also available. Lease Documents if any can be attached.

A lease can be added by clicking Add icon with basic lease information such as Landlord or Tenant Name, Lease Identifier, Lease Property Type as Whole Building or Part of Building, Lease Type, Lease Category, Can be Subleased or not, Can be Renewed or not, Lease Execution Date, Lease Commencement Date, Rent Commencement Date, Lease Expiry Date, Status and any additional data fields defined by the User. The added leases will be in Draft Status initially.

Lease Customer Type can be selected with different options like Lessee & Lessor, Lessee, Lessor, Brokerage Firm etc. by Indus. If Lease Customer Type is selected as Lessee & Lessor or Brokerage Firm by Indus, the Purpose option will be visible in the Add page of lease with options to select Lessor or Lessee.

If the Customer Type set by Indus is Lessee, then the Purpose option will not be displayed and the Landlord dropdown list will be displayed and loaded with existing Landlords and if the Customer Type is Lessor, then the Tenant dropdown list will be displayed and loaded with existing Tenants in the add page of Lease.

For Lessee & Lessor or Brokerage Firm type customer, an option to select the purpose of the lease will be provided. User can select Lessor or Lessee option based on the requirement. If the selected purpose is Lessor, then the Tenant dropdown list will be loaded and if the selected purpose is Lessee, then the Landlord dropdown list will be loaded.

User can enter/select Tenant/Landlord Name, Lease Identifier, Lease Property Type, Lease Type, Lease Category, Can be Subleased or not (enable only when the selected purpose is Lessee), Can be Renewed or not, Lease Execution Date, Lease Commencement Date, Rent Commencement Date, Lease Expiry Date, Status and any additional data fields defined by the User.

If Whole Building is selected as Lease Property Type, then only Building can be selected for Lease. If Part of Building is selected as Lease Property Type, then only Floor can be selected for Lease.

If Lessor is selected as Purpose of the lease, then Lease Types could be Direct Financing Lease, Leveraged Lease, Operating Lease or Sales Type Lease. If Lessee is selected as purpose, then the Lease Types could be Capital Lease or Operating Lease.

Direct Financing Lease: Method used by lessors in capital leases when both of the following criteria for the lessor are satisfied:

(1) Collectability of minimum lease payments is predictable and

(2) No important uncertainties surround the amount of unreimbursable costs yet to be incurred.

In a direct financing lease, the lessor is not a manufacturer or dealer; the lessor purchases the property only for the purpose of leasing it.

Leveraged Lease: Three-party lease contract under which a lessor borrows some or most of the funds to finance the asset to be leased to a lessee.

In this arrangement,

(1) financing provided by the lender is without recourse to the lessor,

(2) the lender holds the title to the leased asset, and

(3) the lessee's payments are assigned to the lender who can repossess the leased asset in case of a default.

Leveraged leases are true (tax oriented) leases because the lessor enjoys all the tax benefits of ownership (such as depreciation) whereas the lessee can claim the full amount of lease payment as expenses.

Sales Type Lease: Accounting by lessor in which one or more of the four criteria required for a capital lease are met and both of the following criteria are satisfied:

(1) collectability of minimum lease payments is predictable and

(2) no important uncertainties surround the amount of unreimbursable costs yet to be incurred.

Capital Lease: A lease is considered to have the economic characteristics of asset ownership. A capital lease would be considered a purchased asset for accounting purposes.

A lease falls into this category if any of the following requirements are met:

1. If the lease life exceeds 75% of the life of the asset.

2. If there is an option to purchase the asset at a "bargain price" at the end of the lease term.

3. If there is a transfer of ownership to the lessee at the end of the lease term.

4. If the present value of the lease payments, discounted at an appropriate discount rate, exceeds 90% of the fair market value of the asset.

Operating Lease: A contract that allows for the use of an asset, but does not convey rights of ownership of the asset. An operating lease is not capitalized; it is accounted for as a rental expense in what is known as "off balance sheet financing". For the lessor, the asset being leased is accounted for as an asset and is depreciated as such. Operating leases have tax incentives and do not result in assets or liabilities being recorded on the lessee's balance sheet, which can improve the lessee's financial ratios.

If Lease Type is Direct Financing Lease, Sales Type Lease or Capital Lease, then a link 'Clauses' will be visible to select the clauses based on the selected Lease Type.

For leasing commercial buildings various Lease Categories are available. They are

Gross Lease: This lease category is also called the "Full Service Gross Lease" where the Landlord pays all of the operating expenses of the building (property taxes, insurance, common area maintenance, janitorial, etc.) for the duration of the term.

Modified Gross Lease: This lease category is also referred to as an "Industrial Gross Lease" where the Landlord pay for one or more of the operating expenses. There is no standard for which expenses are the Landlord’s responsibility. Landlord and Tenant can agree as to who pays for what.

Net Lease: In addition to rent, the Tenant pays for their pro rata share of operating expenses including property taxes, insurance and common area maintenance. In most cases, the Landlord is responsible for the roof, parking lot and possibly the foundation.

Triple Net Lease: This lease category is also known as "NNN Absolute" lease where the Tenant is not only responsible for the operating expenses as in a Net Lease, tenant is obligated to pay for all repairs to the property. The Landlord simply collects rent which, unlike a Gross Lease, represents the Landlord's net operating income. Net leases are very popular with single tenant retail and industrial users. In the Metro Boston market, this is the most popular type of lease.

Percentage Lease: This lease category is also known as "Shopping Center Lease". The Percentage Lease is additional rent due to the Landlord beyond just the base rent. The extra rent is based on sales over a specified amount, called the breakpoint. This type of lease used almost exclusively in retail settings of large shopping centers and malls. It is beneficial to both Tenant and Landlord because the Landlord has a financial stake in the success of the tenants which encourages owners to maintain the property and wisely choose a complementary mix of tenants.

When adding a lease, the status of the lease will be 'Draft'. User can change the status of Lease from 'Draft' to 'Active' by clicking on Edit icon. The status of the lease will be changed by the system automatically as Expired based on the Lease Expiry Date. It will also show as 'Terminated' or 'Deleted' if it is terminated or deleted in between by the User.

Add

A Lease can be added by clickingAdd icon. The added leases will be in Draft status.

Edit

A Lease can be edited by selecting it and then clicking the Edit icon. User can edit all the fields if the selected lease is in Draft status. For Leases in Active status, only editable field is 'Status'. User cannot edit leases in Deleted, Expired and Terminated status.

Delete

A Lease can be deleted by selecting it and then clicking the Delete icon. User can delete Leases in Draft status only.

Area and Cost Details

This option provides users to Add/Edit Area and Cost details for a lease, only if the selected lease is in Draft status. User can view Area and Cost details of a lease if it is in Active, Deleted or Terminated status. User can Add/Edit or View Area and Cost Details through Area and Cost Details menu from icon.

The area selected for the lease will be shown in the Calculated Area field of Area and Cost Details page. User can enter documented area in the Documented Area field.

If the Lease Type of the selected lease is Capital, then some extra fields like Bargain Value Market Value and Economic Life will be visible. Economic Life of a Building will be visible only if Lease Property Type of the selected lease is Building only. Payment Interval will be populated with intervals like Monthly and Yearly. User can enter Lease Amount also.

Economic Life- This refers to the period over which a property is expected to be usable with normal repairs and maintenance. Here Economic Life is usually expressed in number of years.

Payment Interval – User can select Monthly or Yearly as Payment Interval.

Bargain Value – The value that the leased property can be purchased by the lessee at the end of the lease term for an amount significantly less than the value of the property at that time.

Market Value – The value of a property as currently priced in the market place. After entering all the required data, click on Save button to save area and cost details.

Rent Information

This option provides users to Add/Edit Rent Information and Additional Charges against a lease through Rent Information menu from icon, only if the selected lease is in Draft status. User can only view Rent Information of a lease if it is in Active, Deleted or Terminated status.

Annual Base Rent is the lease amount that the Tenant needs to be paid in a year. User can enter Security Deposit, To be paid at (Beginning or End), Interest Rate (%) and Due Date. Annual Base Rent will be displayed in the Grand Total field of the Rent Information. User can save the rent information by clicking on Save button.

Rent Payment

On selecting Rent Payment menu from icon, the payment information of the selected lease which is in Active status will be displayed here. User can update the payment information by clicking on Edit icon. Initially the status of the Rent Payment will be displayed as 'Past Due'. After updating the payment details, the status will be changed automatically to 'Paid'. The payment due alert mail will be sent to contacts set against the selected lease.

Edit: A payment can be edited by selecting it and then clicking the Edit icon. After entering the paid amount click Save Changes button to update the payment details and the status will be changed automatically from 'Past Due' to 'Paid'.

Lease Items

Building and Floor are grouped under Lease Items icon.

Building

This option is used to attach Buildings to a lease. Buildings can be attached to a lease by clicking on Building menu from Lease Items icon. Then Select Lease Area page will be displayed to select the required Floors. Select required floors and click Save button. Once the floors are selected and saved, the gross area of selected floors will be displayed in the Calculated Area field in Area and Cost Details page.

Floor

Floors can be attached to a lease by clicking on Floor menu from Lease Items icon. Then Select Lease Area page will be displayed to select the required Floors. Select required floors and click Save button. Once the floors are selected and saved, the gross area of selected floors will be displayed in the Calculated Area field in Area and Cost Details page.

Clauses

Agreement Clauses and Cancellation Clauses are grouped under Clauses icon.

Agreement Clauses

When clicking on Agreement Clauses from Clauses icon, all the Agreement Clauses added to the lease will be displayed here. User can add agreement clauses to the selected Lease by clicking on Add icon. User can select multiple Agreement Clauses from this list.

Add: User can add Agreement Clauses by clicking on Add When clicking on Add icon, all the Agreement Clauses will be listed in a new window with already added Agreement Clauses as checked. User can select the required Agreement Clauses and click on Save icon. All the selected Agreement Clauses will be added to the Agreement Clause list.

Cancellation Clauses

When clicking on Cancellation Clauses menu from Clauses icon, all the Clauses which are not Service Clauses will be displayed here. User can select multiple Clauses from the list and can enter cost for the Cancellation Clauses. All the selected Clauses will be added against the lease by clicking on Save icon.

Lease Documents

The Administrator can view/add/edit/delete documents to a Lease using this option. Select the required Lease, and then click the Lease Documents icon. Lease Documents can be uploaded for leases in Draft or Active status.

Select a Lease and clicking Lease Documents icon, any documents that are already attached to this lease will be listed. Documents can now be added to the selected lease.

Add: A document can be added by clicking on the Add icon. The New Document window will be displayed. Select the Document Category first, and then select the required file. The document has to be uploaded by clicking on the Save button.

Edit: A document can be edited by clicking on the Edit icon. Select a document and click on the Edit icon. The Edit Document window will be displayed with the current file name. The user can select the document category and the required file. Changes can be saved by clicking the Save Changes button.

Delete: Delete a document by selecting the required file and clicking the Delete icon.

View: The document can be viewed by selecting the file and then clicking the View icon. The selected attachment will be opened in an attachment viewer. View is supported only for image files.

Download: Document can be downloaded and saved in the user's computer by clicking the Download icon.

Lease Renewal

When clicking on Lease Renewal icon, all the renewed leases for the selected lease will be displayed. Only Active leases can be renewed. User can add or delete Lease Renewals. The newly added lease will be in Draft status. The status of the lease will be changed by the system automatically as Active when the previous lease expires. Then the expired lease will be displayed in the Lease Renewal page in Expired status.

Add: A Lease Renewal can be added by clicking Add icon in Lease Renewal page. The added leases will be in Draft status.

Delete: A Lease Renewal can be deleted by selecting it and then clicking the Delete icon in Lease Renewal page. User can delete Leases in Draft status only.
