The reports generated by the system can be viewed using the controls available in the report viewer. The report can be saved to Excel, PDF or RTF format by clicking on Excel, PDF or RTF button.

Real Property Reports are available under Real Property menu in Reports

The following Reports are available in Real Property Module.

Lease Summary

This report shows the list of Active, Deleted, Terminated, and Draft leases. On selecting Lease Summary, a report will be displayed with details like Tenant, Landlord, Lease Type, Lease Category, Lease Commencement Date, Lease Expiry Date, Lease Execution Date, Lease Identifier, Lease Property Type, Rent Commencement Date and Status.

The report can be saved to Excel, PDF and RTF format.

Critical Dates

This report gives the details of Critical Dates of Active leases. On selecting Critical Dates,a report will be displayed with details like Lease Identifier, Commencement Date, Expiry Date, Execution Date and Rent Commencement Date.

The report can be saved to Excel, PDF and RTF format.

Leased Spaces

This report shows the details of leased spaces of all the active leases with details like Lease Identifier, Site, Building, Floor and Calculated Area.

The report can be saved to Excel, PDF and RTF format.

Current Leases

This report shows all the active leases with details like Lease Identifier, Landlord, Tenant, Expiry Date, Documented Area and Lease Amount.

The report can be saved to Excel, PDF and RTF format.

Cost Obligation

This report shows the average cost per area for each lease with details like Lease Identifier, Landlord, Tenant, Expiry Date, Annual Base Rent, Documented Area and Average Cost. The Average cost is calculated by dividing Annual Base Rent with its Documented Area.

The report can be saved to Excel, PDF and RTF format.

Facilities Summary

This report shows the details of buildings for each site with details like Site Name, Number of Owned Buildings, Number of Leased Buildings, Total Number of Buildings, Total Owned Square Footage, Total Leased Square Footage, Total Square Footage and Percent of Total Inventory (%). The Percent of Total Inventory is calculated by dividing Total Square Footage of a site with the Sum of Total Square Footage of all sites, multiplied by hundred.

The report can be saved to Excel, PDF and RTF format.

Lease Renewing in Next Year

This report shows the leases that are to be renewed in the next year. The report includes details like Site Name, Building Name, Building Code, Lease Identifier, Expiry Date, and No. of Days.

The report can be saved to Excel, PDF and RTF format.

Facility by Lease Service Clauses

This report shows the facility (building) by lease service clauses of leases with ‘Lease Property Type’ as ‘Whole Building’. The report includes with details like Service, Service Paid by Lessee, Service Paid by Owner and N/A.

The report can be saved to Excel, PDF and RTF format.

Buildings by Year

This report shows the list of buildings based on the Construction Year and its Ownership type. If Construction Year is not entered then the year in which the building was added will be taken as the construction year. The report shows only the latest 10 years data

The report can be saved to Excel, PDF and RTF format.

Building Area by Year

This report shows the list of the Area of buildings based on the Construction Year and its Ownership type. If Construction Year is not entered then the year in which the building was added will be taken as the construction year. The report shows only the latest 10 years data.

The report can be saved to Excel, PDF and RTF format.

Buildings by Condition

This report shows the condition of the buildings rated in percentage. The report can be saved to Excel, PDF and RTF format.

Renovation Details by Buildings

This report shows the building renovation cost of buildings for last 10 years with details like Site, Building, Code and Renovation date.

The report can be saved to Excel, PDF, Rich Text and Text formats.

Lease Cost Projection for an Year

This report shows the cost projection details (rent and additional charges) for the current year with details like Lease Identifier, Annual Base Rent, Additional Charge, Rent Paid and Rent Due.

The report can be saved to Excel, PDF, Rich Text and Text formats.

Rent Rolls

This report shows the list of Payment information of all the Active Leases in the selected Date period with details like Lease Identifier, Due Date, Lease Amount, Paid On, Paid Amount and Status.

The report can be saved to Excel, PDF, Rich Text and Text formats.
