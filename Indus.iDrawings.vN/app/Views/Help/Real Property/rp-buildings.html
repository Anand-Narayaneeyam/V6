This option is used to display the details of the buildings added through the Administration module.

Owned

This option will list all the Buildings with its Ownership Type assigned as 'Owned' or 'Purchased' with their Site Name, Building Name, Ownership Type, Floor count, Attachments etc. The Gross Area value will be the total Gross Area of all floors in the building. Any additional data fields added for building will also be displayed. The user can add, edit, delete, view or download Attachments to the selected Building. List can be sorted by clicking the column headers. This option allows the users to edit building details and add renovations to the buildings.

Edit: This option is for editing the building details. Select the building to be edited and click Edit icon.The details will be displayed in a new window. Make necessary changes and click the Save Changes button. In Edit Building window, Estimated Useful Life refers to the period over which a building is usable. Here Estimated Useful Life is usually expressed in number of years. Salvage Value refers to the estimated value of property at the end of its useful life.

Renovations: This option is used to view/add/edit/delete Renovations associated to the selected building. Select the Building that requires renovation and click Renovations icon. The user can provide Renovation details like the Renovation Date and Cost invested in the renovation process of the selected Building using this option.

Add: A Renovation can be added by clicking on Add icon.Enter the Renovation Date and Cost. User can save the renovation details by clicking on Save button.

Edit: A Renovation can be edited by selecting it and then clicking the Edit icon. By clicking this option only Cost can be edited.

Delete: A Renovation can be deleted by clicking the Delete icon.

Attachments

Administrators can add/edit/delete/view/download attachments to a building using this option. Select the required Building and click the Attachments icon. The files that can be attached will be restricted to data files like Word, Excel, pdf, jpg, gif, bmp etc.

The File attachments window opens up on selecting Attachments icon, showing any files that are already attached to this building. Attachments can now be added to the selected building.

Add: An attachment can be added by clicking on the Add icon. The New Attachment window will be displayed. Select the Attachment Category first, and then select the required file. The attachment has to be uploaded by clicking on the Upload button.

Edit:An Attachment can be edited by clicking the Edit icon.

Select an attachment and click on the Edit icon. The Edit Attachment window will be displayed with the current file name. The user can select the attachment category and select the required file. The attachment has to be updated by clicking on the Save Changes button.

Delete: Delete an Attachment by selecting it and clicking the Delete icon.

View: The attachment can be viewed by selecting the file and then clicking the View icon.

Download : Attachment can be downloaded and saved in the user's computer by clicking the View icon.

Export

This option is to export the owned building data to the Excel. On clicking Export icon, data will be exported to Excel.

This option allows the users to view the world map and it helps the users to locate the buildings on the map. The pushpins will indicate the building location and are displayed based on the coordinates (latitude and longitude) provided while adding a new building. On placing the cursor on these pushpins, tool tip showing its corresponding building details will be displayed. The user can right click on the pushpin to edit the building details including the coordinates or select "Validate on Map" link in the Edit Building page to reallocate the building location on the map. This option will be available only if "Enable GIS Support" option is enabled by Indus.

The user can zoom in/out the map to trace the Building locations. The Ownership Type combo box displayed at the top of the map page lists the different ownership types such as All, Owned and Leased. Based on this selection the corresponding buildings will be indicated on the map. "All" option will be selected by default in the Ownership Type combo box.
