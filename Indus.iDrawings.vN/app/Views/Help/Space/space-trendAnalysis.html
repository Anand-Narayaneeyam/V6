The trend analysis shows the trends in the Usable Areas, Chargeable Areas, Space Functions and Costs for all the Organizational Units (Divisions, Departments, Cost Centers, etc). This option is available only to the Administrator.

The trends are based on data from the Snapshots saved by the user. Snapshots can be taken periodically or whenever there is a significant change in space assignments.The trends in areas show the value on the Start Date (the Start Date selected by the user), Distribution Percentage for the Organizational Unit on this Date, value on the End Date (the End Date selected by the user), Distribution Percentage for the Organizational Unit on this Date, Average Sq. ft. for the selected period and Average Distribution percentage.

Snapshots

To see the Trends or track changes in the spaces occupied by the Organizational Units (Divisions, Departments, Cost Centers etc.) in an Organization, users need to save Snapshots of data as and when changes happen. It should be a deliberate action taken by the user when he/she feels that a particular space change is significant.

A Snapshot saved on a particular date, will have the Usable Areas, Chargeable Areas, Functional Areas and Costs of Space for all the Organizational Units. These values are in fact the corresponding values available in the Organizational Distribution Report on that date. Snapshots can be exported to select file format from the list.

On clicking Snapshots option, a list of all available snapshots will be displayed; the details of a snapshot can be viewed by selecting the snapshot, and clicking View icon.

Add: On clicking the Add icon , Snapshots can be created and exported to Excel/pdf/txt/rtf files or saved in the database specifying a Snapshot date.

Delete : By selecting one or more snapshots from the list and then clicking the Delete icon, the snapshot can be deleted.

View: The details of a snapshot can be viewed by selecting the snapshot, and clicking View icon.

Chargeable Areas

This report shows the trend in the Chargeable Areas for the Organizational Units over the selected period of time. The chargeable area calculation is based on the user's preference to include or exclude External Wall Areas and/or Vertical Space Areas. The option to make this choice is available in the Chargeback Options under the Space Module Setup. Report can be created for a specific period. The report can be exported to select file formats.

History

A detailed trend report can be seen by clicking the History button on top. This shows the area occupancy report, i.e. the area occupied by the Organizational Units during periods of change.

Usable Areas

This report shows the trend in the Usable Areas for the Organizational Units over the selected period of time. Specify the period for which the report is required. Click on Show Report icon,and the report will be opened. The report can be exported to selected file formats.

Cost

This report shows the trend in the Cost for the Organizational Units over the selected period of time.

Space Functions

This report shows the trend in the Space Areas occupied by the Organizational Units on the basis of Space Function, over the selected period of time.
