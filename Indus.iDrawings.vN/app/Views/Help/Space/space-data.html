This section provides an option for the user to view and analyze the space data graphically.

Click on this, and the option for selecting the floor will be listed. Select the desired floors for which the data is to be viewed and click on the Show Data icon.The details will be listed in the Space data grid.

Edit

Using this option, the administrator can edit the space details. Select the space to be edited and click on the Edit icon.The details will be displayed in a new window. Make necessary changes and click the Save Changes button. Save Changes button is visible only to users having Edit privilege.

The Room Number field will be visible only if it is enabled by Indus and it can be set as Unique for the Organization/Site/Building/Floor. It is also possible to have the Site or Building Code or both as prefix to the Room Number. This is disabled by default, but can be turned on by Indus.

If the Room Number is set to be unique and the entered room number already exists, it will display a message “Room No. already exists for the Organization/Site/Building/Floor” corresponding to the set Uniqueness. If any prefix has been set for Room Number by Indus, then a read-only prefix will be displayed before the Room Number Text box. Only the Room Number need be entered, the prefix will be automatically inserted.

By default, Space Category is enabled. If the value selected is one of the Common Area space categories, like Building Common, Floor Common or Vertical Space it will not be possible to edit/set the Organizational Unit for this space, and all Organizational level combo boxes will be disabled. But if it is required to assign Organizational Units to Common Spaces also, Organizational Unit Assignment to Common Area can be enabled by Indus.

Data for additional data fields can also be entered through this data entry window.

Share Space

Provision for Sharing a Space between Multiple Organization Units is allowed. This is needed for sharing spaces like Consulting Rooms, Conference Halls, etc. between different departments.

If ‘Space Allocation to Multiple Departments’ option is enabled by Indus, a link ‘Share Space’ will be displayed along with Organizational Unit in the Data Entry window. The Administrator can view/add/edit/delete the sharing of space using this option. The Sharing among different Organizational Units should sum up 100.

If the selected Space is already shared, it will be indicated with a caption ‘Shared’ next to the Organizational Unit name. Click ‘View’ link to view the existing sharing details.Click ‘View’ link, the Space Sharing by Organizational Units window opens up with the percentages of already assigned Organizational Units.

Add: On clicking Add icon, Assign Shared Organizational Unit window will be displayed with Org. Units combo boxes and Percentage textbox. Select the required Organization Unit name from the required level and enter the percentage. The sharing details will be saved by clicking Add button.

Edit: Space sharing can be edited by clicking the Edit icon. The Sharing among different Organizational Units should sum up to 100.

Delete: Remove an assigned Organizational Unit by selecting the same and clicking the Delete icon.

Multiple Edit

Multiple records can be edited by selecting the required spaces using the Shift or Ctrl keys, and then clicking on the Edit icon. A window will be displayed listing the fields that can be edited. Select the required Field and enter or select the value to be updated to the selected Spaces and click Save Changes button. The changes have to be made selecting one field at a time.

Assign Space Standard

This option is for assigning space standard and space assignment type to spaces.Select Assign Space Standard icon.Space standard and Space Assignment Type can be assigned/de-assigned by invoking this option and then selecting the space for which space standard is to be assigned. Space Assignment Type can be set to Assignable, Scheduling Space and Unassignable. Scheduling Space option will be available only if Team Room Booking module is enabled for the customer.

Assignable:

If Workspace Booking is enabled for the customer and the Space Assignment Type is selected as Assignable, a link named 'Seats' will be displayed against Room Seating Capacity option to define seats for the selected space. On selecting this link, Seats window will be displayed listing all the available seats (if any).

User can add or delete single/multiple seat(s) to/from the selected space using the following options listed in the Seats page. An option to set seats as Hoteling Seats also will be displayed in the Seats list page.

Different options such as with Prefix, With Room Number and without Prefix are provided to set the Seat Number while adding the Seats.

Add Single Seat: The user can add seats to the selected space, one at a time using this option. To add a single seat to the selected space, the user has to select Add icon and select 'Single' radio button. Then options to specify the Prefix and the Number of the specific seat will be displayed in the Add Seat window. The number of seats that can be added is limited to the Minimum Seating Capacity assigned to the selected space.

Add Multiple Seats: The user can add multiple seats at a time to the selected space using this option. To add multiple seats to the selected space, the user has to select Add icon and select 'Multiple' radio button. Then options specify the Prefix of the seats and the number range in which the seats are to be added in the Add Seat window. The number of seats that can be added is limited to the Minimum Seating Capacity assigned to the selected space.

All the Seats set as Hoteling Workspaces can be booked by Users of all User Roles.

Delete Seat: Seats added can be deleted by selecting the required seat and then clicking Delete icon.

Update: Seats can be set as Hoteling Seat by selecting the checkbox of the required Seat and then clicking Update button.

Hoteling:If Space Assignment Type is selected as Hoteling and a Room Seating Capacity is given, Hoteling Workspaces (equal to the count of Room Seating Capacity) will be added to Space. These Workspaces can be booked for Workspace Reservations. These Workspaces can be booked by Users of all User Roles.

Special Use Rooms: If Space Assignment Type is selected as Special Use Rooms and a Room Seating Capacity is given, Special Use Rooms (equal to the count of Room Seating Capacity) will be added to Space. These Workspaces can be booked for Workspace Reservations by the Concierges only.

Scheduling Space: If Space Assignment Type is selected as Scheduling Space, an option to select the Amenities of the room and its count will be displayed. A Team Room can be booked only if the Space Assignment type is assigned as Scheduling Space.

Totalize

This option is to view the Total Gross Area and Net Area of the selected spaces. The Net Area will be displayed only if the Net Area option is enabled.

Select the records to be totalized and click on Totalize icon. A message box will be displayed with the Number of Spaces selected, and the Gross and Net Areas of the selected spaces.

Apart from this, if any additional data fields of Space are set as Totalizable, the total value of such fields also will be shown.

Attachments

Administrators can add/edit/delete/view/download attachments to a Space using this option. Select the required site and click the Attachments icon. The files that can be attached will be restricted to data files like Word, Excel, pdf, jpg, gif, bmp etc.

The File attachments window opens up on selecting Attachments icon, showing any files that are already attached to this site. Attachments can now be added to the selected site.

Add: An attachment can be added by selecting the Attachment Category first, and then selecting the file through the file open dialog by clicking on the Browse button. The attachment has to be uploaded by clicking on the Upload button.

Edit: Edit an Attachment by selecting it and clicking the Edit icon.

Select an attachment and click Edit icon.The Edit Attachment window will be displayed with the current file name. The user can select the Attachment Category. There is an option to replace the selected attachment with a new file. (It is possible to browse the file location by clicking the Browse button).

Delete: Delete an Attachment by selecting it and clicking the Delete icon.

View : The attachment can be viewed by selecting the file and then clicking the View icon.

Download : Attachment can be downloaded and saved in the user's computer by clicking the Download icon.

Display Settings

This option allows the user to change the visibility of the data fields displayed in the data grid. On selecting this option, all the available data fields are displayed in a window. Display settings can be modified by selecting the required fields from the list. Changes made here will be saved as user default settings and will be effective whenever the User takes the Space Data grid.

For making permanent changes, an administrator has to click Yes to the confirmation message “Display Settings updated. Do you want to save this as System default?”

Show Drawing

Selected space can be viewed by clicking on the record first and then by clicking the Show Drawing icon. The drawing will be displayed with the selected space hatched.

Show Zoomed

Selected space can be viewed as zoomed by clicking on the record first and then by clicking the Show Zoomed icon.The zoomed drawing will be displayed with the selected space hatched.

Export

This option is to export the space to the Excel. On clicking Export icon, a confirmation message, “Do you want to export the data along with Space Handle?” with Yes and No buttons will be displayed. On clicking Yes to the message, data will be exported with space handle. Else it will be exported to excel without space handle.

If any Shared Spaces exist, such spaces will be exported as separate rows for each organizational unit in the shared space. All space details except the area details and organizational units will be same for those rows.

Keyword Search

This option helps users to search for space details using keywords. The keyword could belong to any data field of the space data. The user need not know or specify which field to look in. The Keyword could be any part of the space data viz. space category, space function, room number, cost category, area, etc. or the additional data fields.

On entering the keyword and clicking on the Search icon, the data grid will be filtered with space records having data matching the specified keyword.

A facility user would be able to view only those floors/divisions for which privileges are granted to him.

Advanced Search

This option helps users to search for space details with specific data in known data fields. The data being searched for can be selected from the list boxes or entered into text boxes provided against the data fields. The user can select one or more data fields viz. space category, space function, room number, cost category, area, etc. or the additional data fields. After entering the search criteria and click Search button, the data gird will be filtered with space records which satisfy the selection criteria.

Fields to be displayed in the Search window can be set using the Configure icon, provided in the Search window.

Search criteria entered in the Search window can be cleared by clicking the Clear button provided in the Search window. After clearing the Search criteria and click Search button to clear the Search Results in the Space Data grid.

A facility user would be able to view only those floors for which privileges are granted him.
