This option is available under Settings icon

This option allows the Administrator to delete all the Space Data for the selected drawing. On selecting this option, the floor list is displayed. From this list select the floor for which the space data is to be deleted. Click on the Delete icon , the system prompts with a message ‘Are you sure you want to delete the Space data of the selected floor(s)?’ On clicking ‘Yes’ to this message, it will delete all the space data and save all object details for re-linking later.

The drawing is automatically locked after the Delete Space Data operation.
