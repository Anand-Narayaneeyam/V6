Purpose
To sync employee data
Description
To import employee data to iDraawings from external Apps
Prerequisites
Web service username and password to login to sync data to iDrawings
Base URL
{{baseUrl}}
API Details
API URL
{BaseUrl}/api/IndusExport/ExportEmployeeData
Access Method
POST
Input Class Name
WSEmployeeDataInput
Input Properties of WSEmployeeDataInput
Parameter Data Type Description
UserName String Web service user name to login for updating the data through web API
Password String Web service login password for updating the data through web API
IsDataInsert Boolean To denote whether to Insert or Update the data
  • True to Insert data
  • False to Update data
DataInput JSON The data which is to be ported to iDrawings Database
